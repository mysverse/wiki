## 什么是 MYSverse？

MYSverse 在 Roblox 上打造沉浸式的马来西亚主题体验，将乐趣、
文化与教育价值融入真实的元宇宙环境中。

## 我如何加入角色扮演社群？

MYSverse Sim 的角色扮演机构，例如警察和军事社群，会定期举办招募活动，通常会在我们的 Discord 上公告。
访问 Bandaraya 或 Lebuhraya 页面，了解如何参与商业活动及其他社群组织。

## 有合作机会吗？

当然有！我们非常希望与政府机构和区域品牌合作。
请查看我们的[主网站](https://mysver.se)以获取联系方式。

## 我可以为维基做贡献吗？

当然可以！我们的维基由社群驱动，可通过 GitHub 编辑。
使用页面下方的"在此编辑此页"按钮提交 Pull Request，我们的团队会进行审核。

## 谁可以玩这些体验？

欢迎所有人！虽然我们的内容具有独特的马来西亚特色，但我们打造的
体验面向全球观众，所有人都能享受其中。
