# Bandaraya 中的队伍

Bandaraya 拥有各种各样的队伍，每个队伍都提供独特的游戏体验、车辆、工具和职责。队伍可通过主菜单访问，可能需要加入 Roblox 群组、购买 gamepass 或获得特殊权限。

## 默认队伍

### Visitors

- **简介：** 所有新玩家的默认队伍。无论是出差还是旅游——自由探索城市。
- **解锁条件：** 全部开放（默认队伍）。
- **队伍颜色：** 亮绿色

### Malaysians

- **简介：** 自豪的马来西亚公民。
- **解锁条件：** 加入 MYSverse Malaysian Community Roblox 群组。
- **队伍颜色：** 亮黄色

### High Class Citizen

- **简介：** 拥有专属权限的高级平民角色。
- **解锁条件：** 特殊要求（gamepass 或群组等级）。
- **队伍颜色：** 深橙色

## 执法队伍

这些队伍专注于维持秩序、应对事件和在全城执行法律。

### Polis MYSverse (POLIS)

- **简介：** MYSverse 警察——Bandaraya 的主要执法机构。运营多个部门，包括常规巡逻、刑事调查部（CID）、交通警察（JSPT）、特别行动部队（UTK）和装甲部队（GGK）。
- **解锁条件：** MYSverse Sim Police 群组成员资格或相关 gamepass。
- **队伍颜色：** 亮蓝色
- **设施：** IPD Dang Wangi CID 办公室、IPD Dang Wangi JSPT 办公室、Balai Polis Kuala Lumpur
- **车辆：** 常规巡逻车辆、CID 便衣车辆、JSPT 交通车辆、GGK 战术车辆、运输车辆、旋翼飞行器（直升机）
- **工具：** 枪械、电击枪、手铐、测速枪、平板电脑（车辆登记查询）、无线电、锥筒、标志牌

### Polis Bantuan MyTransit

- **简介：** 分配至 MyTransit 公共交通运营的辅助警察部队。
- **解锁条件：** 需要群组成员资格。
- **队伍颜色：** 海军蓝

### Kor Polis Tentera MYSverse (KPTD)

- **简介：** MYSverse 宪兵队——负责执行军事法律和纪律。
- **解锁条件：** 需要群组成员资格。
- **队伍颜色：** 亮红色
- **车辆：** 专属 KPTD 车辆

## 军事队伍

### MYSverse Armed Forces (MAF)

- **简介：** MYSverse 武装力量。从 Kem Wardieburn 军营运作。
- **解锁条件：** 需要群组成员资格。
- **队伍颜色：** 军绿色
- **设施：** Kem Wardieburn
- **车辆：** 轻型车辆、装甲车辆、民用车辆

### Royal Marine Corps (RMC)

- **简介：** 外国盟军海军陆战队。
- **解锁条件：** 需要群组成员资格。
- **队伍颜色：** 中蓝色
- **车辆：** 专属 RMC 车辆

## 紧急服务

### Bomba MYSverse（消防与救援）

- **简介：** MYSverse 消防与救援部门。在全城应对火灾事件，操作消防车，并使用消防栓系统。
- **解锁条件：** 需要群组成员资格或 gamepass。
- **队伍颜色：** 栗色
- **设施：** Central Fire Station Kuala Lumpur
- **车辆：** 消防救援车、工具车、直升机
- **工具：** 消防水带（供水管线 + 攻击管线）、消防栓连接、无线电、警戒带
- **游戏玩法：** 火灾会在城市中随机出现。将供水管线从消防车连接到消防栓，然后使用攻击管线扑灭火灾。

### Kementerian Kesihatan MYSverse（卫生部 / KKM）

- **简介：** 国家卫生部——作为医疗人员拯救生命。
- **解锁条件：** 需要群组成员资格。
- **队伍颜色：** 机构白
- **设施：** KKM Ampang Clinic、Prince Court Medical Centre
- **车辆：** 救护车、快速响应车辆
- **工具：** 医疗设备

### St. John Ambulans MYSverse (SJAM)

- **简介：** 志愿救护车和急救服务。
- **解锁条件：** 需要群组成员资格。
- **队伍颜色：** 森林绿
- **设施：** SJAM 地标
- **车辆：** 救护车辆

### RELA Corps

- **简介：** 人民志愿团——支持公共安全的平民志愿力量。
- **解锁条件：** 需要群组成员资格。
- **队伍颜色：** New Yeller

## 其他队伍

### Organised Crime Group (OCG)

- **简介：** 地下犯罪组织。参与非法活动、抢劫和黑市交易。
- **解锁条件：** 需要特殊权限。
- **队伍颜色：** Alder
- **车辆：** OCG 车辆

### Developers

- **简介：** MYSverse 开发团队。
- **解锁条件：** 需要开发者身份。
- **队伍颜色：** 钢蓝色

### Moderation Team

- **简介：** 游戏内管理员，确保安全公平的游戏体验。
- **解锁条件：** 需要管理员身份。
- **队伍颜色：** 品红色

### Arrested

- **简介：** 被执法人员逮捕的玩家。被逮捕后您将被分配到此队伍，需等待刑期结束。
- **队伍颜色：** 亮紫色

## 队伍机制

- **队伍选择：** 通过主菜单的**队伍**选项卡更换队伍。
- **队伍车辆：** 每个队伍在各自设施都有专用车辆生成点。只有相应队伍的成员才能使用这些生成点。
- **紧急分类：** MAF、KPTD、POLIS、Polis Bantuan、Bomba、SJAM、RELA、Developers、KKM 和 Moderation 队伍被归类为紧急/优先队伍。
- **交通分类：** TransitKL、Fetch 和 Flaxim 被归类为工作/交通队伍。
