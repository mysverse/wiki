# 紧急服务

扮演公共服务角色 — 警察、消防、医疗、军人 — 是 Bandaraya 角色扮演的核心。本页介绍每个团队的职责、所配备的工具,以及对穿着制服的人员的期望。

## 简要概览

| 团队 | 您的职责 | 逮捕权 | 主要工具 |
| --- | --- | --- | --- |
| **PDRM** | 巡逻、应对犯罪、拘留、逮捕 | 有 | X26P 电击枪、手铐、saman、指纹扫描器 |
| **BOMBA** | 灭火、医疗搬运、救援 | 无 | 消防水带、探照灯、搬运 |
| **KKM** | 治疗受伤玩家、医院值班 | 无 | 医疗 placeables、手电筒 |
| **SJAM** | 志愿救护车 | 无 | 仅手电筒(基本) |
| **RELA** | 志愿警务支援 | 有限 | 木制警棍、搜查、手电筒 |
| **MAF / KPTD** | 军事、阅兵、礼仪;军警逮捕 | KPTD:有 | 各类军事装备 |

## PDRM(Polis Diraja Malaysia)— 警察

组别:1182710、3048630。任何职级。

### 您的职责

- 在 KL 以有标识的单位巡逻(Civic MPV、Inspira、X70、Hilux)。
- 响应 999 电话(火灾、犯罪、意外)。
- 拘留和逮捕嫌疑人。参见[逮捕与法律](/bandaraya/arrest-and-law)。
- 对交通违规开出 saman(罚单)。
- 使用 **MDT 平板应用**来检查 BG、运行 BOLO、提醒其他警员。
- 在 **IPD Dang Wangi** 值班 — 主要警察局(CID、JSPT、Ops Room、Armoury)。

### 工具

| 工具 | 用途 |
| --- | --- |
| **X26P** | 电击枪 — 非致命性制服嫌疑人 |
| **T-Baton** | 近战武器 |
| **Cuff** | 给被拘留的嫌疑人戴上手铐 |
| **Search** | 向玩家请求搜查同意 |
| **Saman** | 开出罚单 / 罚款 |
| **CL2** | 胡椒喷雾 |
| **RadarGun** | 速度检测 |
| **Torchlight** | 照亮黑暗区域 |
| **FingerPrint Scanner** | 身份验证 |
| **Clamp** | 对违规停放的车辆进行车轮锁 |
| **Traffic Wand** | 指挥交通 |
| **Whistle** | 交通管制 / 警示 |
| **Police Tape** | 封锁现场 |
| **PDRM Placeables** | 锥筒、标志、路障 |

### 车辆(70+)

PDRM 拥有最大的车队:

- **巡逻 / MPV** — Proton Preve、Proton Inspira、Honda Civic、Proton X70、Toyota Hilux 各型、Perodua Aruz/Bezza
- **HPV**(高性能)— Civic Type R、Outlander、CX-5、Lancer Evo X
- **护送** — Inspira 1.8、Civic Type R、Volvo S90 T5
- **UTK**(特别战术)— Toyota Hilux UTK、BMW X5 UTK、IAG Jaws UTK
- **CID** — CID K9、CID Hilux、CID Bike
- **无标识** — Saga、Inspira、Civic、Preve、Waja、Persona、X70、Outlander、Bezza
- **运输** — Hiace 各型、HICOM Handalan II、Hilux 货运
- **拖车** — Isuzu Tow Truck、Hino/Fuso Tow
- **防暴** — Scania P310 Riot、Scania VGF1955 FRU、Handalan II Riot Command
- **摩托车** — PDRM Bike 1–5、UTK Bike、BMW R1200GS、Kawasaki 300 URB
- **装甲** — **AV4** 和 **V150**(防弹)

### 直升机

PDRM 拥有 **PDRMHeliSpawner** — 空中警务。从警局 ViewportFrame 选择器 spawn。

### 期望

- **作为 PDRM 变节是严重违规行为。** 警察犯罪 = ban。
- 遵循角色扮演协议 — 表明身份、说明罪名、给嫌疑人服从的机会。
- 使用平板 MDT 对嫌疑人发出 BOLO;其他警员将收到 badge + 声音 + toast 警报。

参见[逮捕与法律](/bandaraya/arrest-and-law)了解完整的逮捕流程。

## BOMBA(Jabatan Bomba dan Penyelamat)— 消防与救援

组别:2957304。任何职级。

### 您的职责

- 响应在 KL 各处 spawn 的**火灾事件**(10 分钟事件,直到扑灭)。
- 操作**消防栓**和**消防水带**。
- **搬运**受伤的平民或消防同僚。
- 对烟雾弥漫的现场使用**热成像相机**。
- 在 BOMBA 站值班。

### 工具

| 工具 | 用途 |
| --- | --- |
| **Searchlight** | 照亮现场 |
| **Bomba Placeables** | 事件现场的锥筒、标志、路障 |
| **Bomba Tape** | 封锁现场 |
| **Traffic Wand** | 在现场指挥交通 |
| **Carry** | 搬起倒下的平民 |
| **Fire Hose**(连接到消防栓) | 扑灭火灾 |

### 车辆(14+)

- **消防车**:P310 FRT、94D LFRT、Mercedes-Atego LFRT
- **专用**:P310 Hazmat、P310 ALP、Hiace MTV(EMRS)
- **摩托车**:BMW R1250RT(MOTO FIRE、MOTO MEDIC)
- **运输**:HICOM Handalan II Transport、HICOM + Sampan(船)、Mitsubishi Triton Emergency、Westar Maxus V80 Forensic、STORM Hilux
- **多用途**:Mazda CX-5 2022、Triton Utility

### 直升机

**BOMBAHeliSpawner** — 空中灭火 / 救援。

### 火灾事件

当**至少有一位 BOMBA 成员在线**时,火灾会在地图各处 spawn。当您在 BOMBA 时,您的客户端会收到警报(地图图钉 + 声音 + 平板通知)。开车到事件现场,让水流到火焰方块上,将其扑灭。

- **时长上限**:每个事件 10 分钟 — 如果在此时间内未扑灭,事件将被强制清除。
- **Cooldown**:事件结束后 5 分钟才能 spawn 新的事件。
- **需要 BOMBA 在线**:如果所有 BOMBA 在事件中途离开值班,火灾会自动被清除。
- **后加入者**:如果您在事件中途切换到 BOMBA,您的客户端会向 server 询问当前状态,地图图钉会立即显示。
- **地图上有 5 个消防栓** — 策略性地定位自己。
- **Auto-sign** — 加入 BOMBA 会自动为您打卡进行值班追踪(每个 session 一个 Discord webhook)。

## KKM(Kementerian Kesihatan)— 卫生部

组别:5760632。任何职级。

### 您的职责

- 治疗**倒下 / 受伤的玩家**。当任何医疗团队(KKM 或 SJAM)在值班时,全局 bleedout 计时器为 5 分钟(给予医护人员到达时间),被动恢复被暂停。
- 在 **KKM Ampang 诊所**和**鹰阁医疗中心**值班。
- 使用医疗套件物品处理伤势(绷带、吗啡、止血带、血袋、IV 导管、O2 等)。
- 通过 **KKM Logger** 平板应用记录医疗操作。

### 工具

| 工具 | 用途 |
| --- | --- |
| **Torchlight** | 照明 |
| **KKM Placeables** | 医疗现场 placeables |
| **Traffic Wand** | 在现场 |
| **Carry** | 移动受伤玩家 |
| **完整医疗套件**(来自 armoury) | 参见[战斗与健康](/bandaraya/combat-and-health#medical-kit-items) |

### 车辆

KKM 拥有救护车、RRV(快速响应车辆)和多用途车辆。包括 KKM Mercedes Sprinter、Iveco Ambulance、RRV Honda City、RRV Proton Inspira、Toyota Hiace Passenger Van。

### 治疗玩家

1. 抵达倒下的玩家处。
2. 使用套件物品 — 绷带(止血)、吗啡(止痛)、血袋(失血)等。
3. 如果他们已 flatline,使用 **defib**。
4. 他们的状态会随着时间恢复。

参见[战斗与健康](/bandaraya/combat-and-health)了解完整的套件分解。

## 联合行动

紧急服务经常协同工作:

- 重大事件的**多车辆响应** — PDRM + BOMBA + KKM。
- **BOMBA 呼叫**触发 `IncidentEvent`,BOMBA 客户端监听该事件;其他人可以自行出动。
- 来自 PDRM 的 **MDT BOLO** 通知所有值班警员(声音 + toast + badge)。
- **KKM** 稳定倒下的平民;PDRM 可以在稳定后继续他们的逮捕。

## 提示与陷阱

- **按照您的团队扮演。** PDRM 不开第一枪;BOMBA 不逮捕;KKM 不与嫌疑人交战。
- **变节 = 严重 ban。** 在任何公共服务团队上,犯罪 = 立即管理处分。参见[规则](/bandaraya/rules#severe-violations)。
- **使用您的平板。** MDT、MyEmergency、车辆管理、MyTow、KKM Logger — 全部为免费工具。参见[平板](/bandaraya/tablet)。
- **Armour = Armor 部件。** 内部有 `Armor` 部件的车辆是防弹的(例如 AV4)。参见[车辆](/bandaraya/vehicles#bulletproofing)。
- **Spawner 受团队限制。** 如果您的团队无法访问某个 spawner,UI 就不会显示。
