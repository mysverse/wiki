# Bandaraya 的商店与经济

Bandaraya 拥有丰富的游戏内经济系统，以 **Bandaraya Ringgit (BR)** 货币为核心。通过工作赚取 BR，在商店消费，并通过游戏内银行管理您的财务。

## 赚取 BR

### 工作与收入

| 工作 | 所需队伍 | 简介 |
|---|---|---|
| Pos Nasional | 无（任何队伍） | 在邮局配送包裹。通过晋升解锁新的配送车辆。 |
| Taxi Driver | Taxi（需要 gamepass） | 在城市中载送玩家以赚取车费收入。 |
| 队伍薪资 | 机构队伍 | 根据您当前队伍（Police、Military、Bomba 等）定期发放薪资。 |

### 其他收入

- **犯罪活动：** 银行抢劫和其他非法活动可获得大量 BR 奖励，但风险极高。
- **转账：** 其他玩家可以通过游戏内手机向您发送 BR。

## Gamepass（Robux）

Gamepass 通过主菜单的**商店**选项卡使用 Robux 购买，提供永久性加成。

### 主要 Gamepass

| Gamepass | 简介 |
|---|---|
| **Taxi Driver** | 加入 Taxi 队伍，通过载送乘客赚取车费。 |
| **Firearms Licence** | 从枪械商人处购买枪械。 |
| **Boombox** | 为自己和附近的玩家播放音乐。 |

::: tip 跨游戏权益
部分 gamepass 可在 MYSverse 的多个游戏中通用。例如，Taxi Driver gamepass 可能同时在 Bandaraya 和 Lebuhraya 中有效。
:::

## 新玩家提示

1. **从 Pos Nasional 开始** — 无需特定队伍，是赚取第一笔 BR 最简单的方式。
2. **尽早购买 Toolkit** — 仅需 BR 100，对保持车辆正常运行至关重要。
3. **善用小地图** — 商店位置会在地图上标记，方便寻找。
4. **存款到 Vastrix** — 存入多余的 BR 以确保安全。
5. **查看主菜单商店** — 新的 gamepass 和物品会定期更新。
