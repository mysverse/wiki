# 术语表 — 马来西亚用语与游戏缩写

Bandaraya 大量使用真实的马来西亚名称。如果你不是来自马来西亚(或还没见过这些缩写),就在这里查一查。

## 货币与金钱

| 术语 | 含义 |
| --- | --- |
| **BR** | **Bandar Ringgit** — 游戏内货币。可以把它当作游戏里的令吉(RM)。 |
| **CAG** | **Click And Go** — 游戏内储值交通卡(Bandaraya 版本的马来西亚 Touch n Go)。 |

## 组织(队伍)

| 缩写 | 全名 | 是什么 |
| --- | --- | --- |
| **PDRM** | Polis Diraja Malaysia | 马来西亚皇家警察。 |
| **BOMBA** | Jabatan Bomba dan Penyelamat | 消防与拯救局。 |
| **KKM** | Kementerian Kesihatan Malaysia | 卫生部(医生、医护人员)。 |
| **SJAM** | St. John Ambulans Malaysia | 志愿救护车服务。 |
| **RELA** | Ikatan Relawan Rakyat | 志愿警察队。 |
| **MAF** | Malaysian Armed Forces | 军队。 |
| **KPTD** | Kor Polis Tentera Diraja | 皇家宪兵(军警部门)。 |
| **JIM** | Jabatan Imigresen Malaysia | 移民局。 |
| **NWS** | National Wire Services | 游戏内的州级新闻社。 |
| **OCG** | Organised Crime Group | 犯罪角色扮演队伍。 |
| **HCC** | High Class Citizen | 付费高级队伍(gamepass)。 |
| **PB** | Polis Bantuan | 交通辅警(MyTransit)。 |
| **MyTransit** | — | 游戏内巴士营运商(主要的那家)。 |
| **Fermuda / FirstGroup / Flaxim / Fetch / Tompang** | — | 其他游戏内交通 / 电召车公司。 |
| **SPAD** | Suruhanjaya Pengangkutan Awam Darat | 陆路公共交通委员会 — 游戏内负责发出巴士营运商执照。 |

## 地点

| 术语 | 含义 |
| --- | --- |
| **IPD** | **Ibu Pejabat Polis Daerah** — 县警察总部。IPD Dang Wangi 是游戏内的主要警察局。 |
| **Kem Wardieburn** | MAF 军营。 |
| **KLBT** / **Bandar Bus Terminal** | 吉隆坡巴士总站。 |
| **Hazbank** | 游戏内银行(是的,你可以尝试抢劫它)。 |

## 游戏玩法与系统

| 术语 | 含义 |
| --- | --- |
| **LTAA** | **Leave To Avoid Arrest** — 被拘留时断线。你回来后会被自动逮捕 3 分钟。别这么做。 |
| **RTAA** | **Reset To Avoid Arrest** — 使用重置按钮逃脱。为阻止此行为,重置按钮会在 75 HP 或以下时被禁用。 |
| **Saman** | 马来语,意为 **传票 / 罚单**。警察会开出。乖乖缴付。 |
| **BOLO** | **Be On the LookOut** — 针对嫌疑人发出的警方通缉警报。 |
| **MDT** | **Mobile Data Terminal** — 警察平板应用程序,具备 BG Check、BOLO 清单及警报功能。 |
| **TapNGo** | 游戏内高速公路收费站(现金 15 BR / 使用 CAG 卡 12 BR)。 |
| **Ammulaysia** | 持牌枪械经销商。需要 Firearms Licence gamepass。 |
| **MRT** | **Mass Rapid Transit** — 加影线快速通行列车。 |
| **ACS** | **Advanced Combat System** — 武器引擎(处理射击、装填、伤害)。 |
| **Saude** | 葡萄牙语,意为 "health"(健康)。这是血量系统的内部名称。你在 UI 里不会看到这个字,但知道一下也好。 |
