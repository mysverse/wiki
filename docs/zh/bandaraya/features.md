# Bandaraya 游戏功能

Bandaraya 是一个全面的马来西亚城市角色扮演体验，拥有相互关联的各种系统。以下是主要游戏功能。

## 货币与经济

### Bandaraya Ringgit (BR)

- BR 是游戏内的主要货币，用于购买物品、车辆和服务。
- 通过工作赚取 BR，例如 Pos Nasional（邮局配送）、出租车驾驶和基于队伍的薪资。
- 收入根据您的当前工作和队伍定期发放。
- 通过游戏内手机向其他玩家转账 BR。

### Vastrix 银行

- Vastrix ATM 遍布全城各处。
- 以 BR 1,000、BR 5,000、BR 10,000 或 BR 100,000 的增量将 BR 存入您的 Vastrix 账户。
- 使用 Vastrix 账户进行安全储蓄和交易。

### HazBank

- HazBank 是城市中的重要金融地标。
- 该银行可能成为抢劫事件的目标（见下文犯罪活动）。

## 车辆与经销商

### 车辆经销商

Bandaraya 设有车辆经销商，您可以使用 BR 租赁或购买车辆。可用车辆包括广泛的马来西亚和国际品牌：

**马来西亚品牌：**

- **Proton：** Saga、Saga 2022、Inspira、Persona、Preve、Waja、Exora、X50、X70
- **Perodua：** Myvi、Myvi 2020、Bezza、Ativa、Aruz

**国际品牌：**

- **Honda：** Civic、Civic Type R、City
- **Toyota：** Camry、Hilux、Vios、Vellfire
- **Mitsubishi：** Outlander、Lancer Evo X
- **其他：** Ford Mustang、Volvo S90、Nissan R35 GT-R、BMW i7、Mercedes C300、Volvo XC90

**摩托车：**

- Honda EX5、Honda CBR1000RR

每辆车辆可在经销商处选择不同颜色进行自定义。

### 车辆机制

- 车辆可能发生故障，需要使用**工具箱**（BR 100）进行维修。
- 可使用**油桶**（BR 800）为车辆加油。
- 故障车辆可使用拖车服务。
- 车辆具有逼真的驾驶物理效果，包括车头灯、转向灯和碰撞系统。

### 执法工具

- **电击枪：** 警察使用的非致命制服工具。
- **手铐：** 拘留和逮捕嫌疑人。
- **测速枪：** 检测超速车辆。
- **平板电脑：** 通过车牌号查询车辆登记信息。
- **无线电 (Voxel)：** POLIS 和 Bomba 的队伍通讯系统。

## 消防系统

- **火灾事件：** 当 Bomba 队伍成员在场时，火灾会在城市中动态生成。
- **消防栓系统：** 消防栓遍布全城，用于连接供水管线。
- **灭火流程：**
  1. 驾驶消防救援车（FRT）前往火灾地点。
  2. 将**供水管线**从 FRT 连接到附近的消防栓。
  3. 使用**攻击管线**（消防水带）扑灭火灾。
- **装备：** Bomba 成员可使用消防水带、供水管线和实用工具。

## 逮捕与法律系统

### 逮捕机制

- 执法人员可以逮捕犯罪的玩家。
- 被逮捕的玩家将被转移到 **Arrested** 队伍，必须等待刑期结束。
- 专用的 **ArrestedGUI** 会显示逮捕信息，包括逮捕警员、释放时间和案件编号。

### 传票系统

- 警员可以向玩家发出**传票**（交通/法律罚单）。
- **SummonsView** 界面允许玩家查看传票详情。

### 搜身

- 执法人员可以使用 **Searchee** 系统对玩家进行搜查。
- 玩家会被提示同意或拒绝搜查。

### 刑法典

- 游戏设有完善的刑法典系统，明确规定了各种违法行为和处罚。

## 角色扮演功能

### 动作动画

通过屏幕上的控制按钮访问动画菜单，可执行以下动作：

- Sit、Chilling、Injured、Surrender、Squat、Crossed Arms、Leaning 和 Stop。

### 互动元素

- **钢琴：** 城市中可弹奏的钢琴，配有完整的键盘界面、移调和音量控制。
- **电梯系统：** 建筑中的功能性电梯。
- **座位：** 可以在车辆、椅子和城市各处的长椅上就座。
- **放置系统：** 在指定区域放置家具和物品。

### 通讯

- **游戏内手机：** 呼叫出租车、向玩家发送消息、请求紧急服务、管理车辆和转账 BR。
- **无线电系统：** 紧急服务（POLIS、Bomba）的基于队伍的无线电通讯。
- **打字气泡：** 当您正在打字时，角色头顶会出现视觉提示。

### 天气与环境

- **昼夜交替：** 动态时间循环，光照条件随之变化。
- **雨天系统：** 动态天气，带有降雨效果。
- **城市音效：** 环境城市音景。

## 犯罪活动

### 银行抢劫

- 玩家可以尝试抢劫 HazBank，体验高风险、高回报的玩法。
- 执法队伍将对抢劫警报做出响应。

### 非法商贩

- **Hacker NPC：** 以 BR 5,000 购买黑客设备。
- **枪械商人：** 购买非法枪械（需要 Firearms Licence gamepass）。

### 警报系统

- 当事件发生时（抢劫、火灾、紧急情况），紧急警报会广播给相关队伍。
- 警报会在小地图上显示，供响应队伍查看。

## 竞技模式 (HardPoint)

- 基于队伍的竞技游戏模式，队伍争夺控制点。
- 设有计分系统，分为主队和客队。
- 通过 HardPoint 界面显示公告和积分追踪。

## 地图与导航

### 小地图

- 屏幕角落的动态小地图显示您的位置、附近的兴趣点和活跃事件。
- 可以切换和展开小地图以获得更广阔的视野。

### 完整地图

- 通过顶部栏访问完整地图，获得城市的全面概览。
- 可以缩放和拖动进行导航。
- 图标显示车辆、包裹、举报、出租车请求、拖车请求、警报和玩家位置。

### 地点标签

- 当您在城市中移动时，屏幕上会显示当前街道/区域名称。
