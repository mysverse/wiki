# 社交功能

Lebuhraya 设有几个旨在让你与他人一起游玩时体验更佳的系统 —— **Convoy** 系统让大家一起赚取奖励、**999 Emergency** 系统提供由玩家驱动的角色扮演、**Phone** 用于应用程序和通讯,以及 **Tokens** 提供临时加成。

## Convoy

一起开车,赚取更多。Convoy 通过可叠加的 XP 倍率奖励协调一致的游玩。

### 创建 convoy

1. 打开 convoy 菜单(从 Main Menu 或 topbar)。
2. 选择 **Create convoy** —— 你将成为队长。
3. 邀请特定玩家,或将 convoy 设为公开,让 500 stud 范围内的其他玩家申请加入。

### 加入 convoy

- **通过邀请**:队长向你发送邀请;请在 30 秒内接受。
- **通过申请**:打开 convoy 菜单,查看 500 stud 范围内的公开 convoy,并申请加入。队长有 30 秒接受。

### Convoy 规则

- **最大规模**:6 名成员
- **距离**:你必须保持在另一名 convoy 成员的 **200 stud** 范围内才能享有加成
- **队长**:设定邀请规则并可移除成员

### 加成如何叠加

Convoy 加成会加入你现有的 XP 倍率叠加(参阅 [经济与 XP](/lebuhraya/economy#the-xp-multiplier-stack))。

| 项目 | 数值 |
|---|---|
| Convoy 基础加成 | +15% |
| 每多一名成员(两人之后) | 各 +5% |
| Convoy 中的好友 | +10% |
| Convoy 中同队成员 | +10% |
| 连续加成(组队时长) | 每分钟 +2%,上限 +10% |
| **Convoy 总加成上限** | **+50%** |

Convoy 车辆中的乘客获得驾驶者一半的加成。

一个配置完善的 6 人 convoy(好友、同队、共同驾驶 5 分钟以上)能稳定达到 +50% 上限。

## 999 紧急系统

Lebuhraya 的紧急服务由玩家驱动。任何玩家都能拨打 999;紧急队伍玩家会响应。

### 拨打 999

1. 打开你的 **Phone**(topbar 图标)。
2. 点击 **999** 应用。
3. 选择事件类型:
   - **Car Crash**
   - **Injured Person**
4. 你的呼叫会广播给所有当值的紧急救援人员。

每个进行中的事件会显示在救援人员的 pager 上,附上你的位置。

### 响应(紧急队伍)

如果你属于 **Bomba**、**Polis Trafik**、**Undercover Police** 或 **Kesihatan**:

1. 装备你的 **Pager** 工具。
2. 查看进行中的事件面板。
3. 点击接受事件 —— 它将成为你被指派的案件。
4. 驶向现场。
5. 根据事件类型予以处理:
   - **Car Crash** —— 使用 spreader/chainsaw 解救被困驾驶者(Bomba)、处理伤势(Kesihatan)、管控现场(Polis)。
   - **Injured Person** —— 稳定伤情并送往医院(Kesihatan)。
6. 一旦你抵达现场且最低响应时间(**30 秒**)已过,处理事件即可获得奖励。

### 限制

- 同一时间最多可存在 **10 个进行中事件**。
- 拨打者重复拨打 999 有冷却时间。
- 若你在事件进行中更换队伍,你的指派会被取消。

## Phone

每位玩家都有一部可从 topbar 取用的手机。

### 内建功能

- **999 Emergency**
- **GPS navigation** —— 设置前往任何 POI 的路标
- **系统时钟**显示游戏内时间

### 应用程序

- **Rafahia App v3.0** —— MYSverse 社群的统一社交应用(讯息、个人档案、跨体验功能)。

随着 MYSverse 平台新增功能,日后可能会出现更多手机应用。

## Token 与加成

部分临时优势来自 **token** —— 你库存中可消耗的加成物品。

| Token | 效果 |
|---|---|
| **2× XP Boost** | 在指定时长内将你的 XP 倍率加倍 |
| **2× Season Pass Boost** | 将 Season Pass XP 进度加倍(在赛季进行中时) |
| **Team Access Pass** | 在不拥有 gamepass 的情况下,临时获得 gamepass 限定队伍的访问权 |

### 如何取得 token

- 职业等级奖励(在某分支升级)
- Season Pass 等级奖励
- 促销活动
- 有时与 gamepass 一同捆绑

### 如何使用 token

打开 **Active Tokens HUD**(当至少有一个 token 启用时,会出现在你 HUD 上的 ScreenGui)。点击库存中的 token 启用它 —— 时长立即开始倒数,并应用于你接下来获得的 XP。

> Token 在**启用时即被消耗** —— 一旦启动,你无法暂停 token。请保留至你能持续赚取奖励的时段使用。
