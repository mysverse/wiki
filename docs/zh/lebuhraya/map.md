# 世界与地图

Lebuhraya 设于一段马来西亚高速公路上，三个主要区域通过收费站和休息站相连。本页将标示出所有地点的位置。

## 各区域

### Bandar Seri Putra

最大的聚落——你进行工作、服务和商业活动的主要中心。如果你是新玩家，头几个小时大多会在这里度过。

**主要建筑**：
- **Balai Polis Bandar Seri Putra** — 警察局
- **Balai Bomba Bandar Seri Putra** — 消防局
- **Klinik Kesihatan Bandar Seri Putra** — 健康诊所 / 医院
- **Depoh Bas Bandar Seri Putra** — 巴士车站（巴士司机路线起点）
- **Radtrol Petrol Station Bandar Seri Putra** — 加油
- **Stesen Insinerasi Bandar Seri Putra** — 垃圾处理投放点
- **Gravstone Workshop Bandar Seri Putra** — 车辆维修
- **Pangsapuri Seri Putra** — 公寓大楼 / 房屋地段
- **Masjid Bandar Seri Putra** — 清真寺
- **KTN Bandar Seri Putra** — KTN 分行
- **Dus Doffee Bandar Seri Putra** — 咖啡店

### Jersik

一个拥有工业区和赛道的次要城镇。

**主要建筑**：
- **Balai Bomba Jersik** — 消防局
- **Pos Nasional Jersik** — 邮政局（邮政局工作起点）
- **Plaza Tol Jersik** — 收费站
- **Jersik Race Track** — 主要赛车场
- **Stesen Insinerasi Jersik** — 垃圾处理投放点
- **H Plantation Jersik** — 农业庄园（伐木工 / 林业）
- **Kompleks Petroliam dan Gas Radtrol** — 大型加油综合设施

*Jersik 没有自己的警察局——最近的 Balai Polis 位于 Kampung Merbang。*

### Kampung Merbang

一个较宁静的乡村。

**主要建筑**：
- **Balai Polis Kampung Merbang** — 警察局
- **Masjid Kampung Merbang** — 清真寺
- 附近的高速公路休息站（见下文）

### Taman Mahkota / Putra Mahkota

一个广场 / 住宅区。

**主要建筑**：
- **Plaza Tol Putra Mahkota** — 第二个主要收费站
- **Taman Mahkota** — 房屋 / 住宅区

## 高速公路地标

高速公路连接三个区域。重要停靠点：

- **PETROMAS Petrol Station Northbound** — 北向加油站
- **PETROMAS Petrol Station Southbound** — 南向加油站
- **RnR Sungai Merbang Northbound** — 提供食物的休闲休息站
- **RnR Sungai Merbang Southbound** — 提供食物的休闲休息站
- **Plaza Tol Jersik** — 收费站
- **Plaza Tol Putra Mahkota** — 收费站

## 天气与时间

Lebuhraya 有实时昼夜循环和随机降雨。

**时间循环**：游戏内时间以**每现实秒推进 1 分钟**的速度运行，因此游戏内完整一天需要 24 分钟现实时间。街灯和广告牌灯在**下午 6:00** 自动开启，并持续亮至**早上 6:40**。在这些时段请开启大灯行驶以确保视野——并尊重氛围；某些道路在夜晚会显著变暗。

**降雨**：阵雨会随机出现。平均而言，每**30–60 分钟**下雨一次，持续**5–10 分钟**。雨下的时候你会听到雨声并看到屏幕上的雨效果。道路不会变滑，但能见度下降。管理员可以在活动期间通过聊天指令 `!rainon` / `!rainoff` 开关降雨。

## 交通灯与路口

路口的交通灯在多个信号组之间协调（例如 Bandar Seri Putra 的 `BSP_Main` 路口有两个交替阶段）。独立的交通灯独立循环 绿 → 黄 → 红。红灯停车由反作弊系统执行——详情请参阅下方*测速和收费站*关于 **Red Light** 违规的说明。

## 测速和收费站

部分高速公路路段设有**测速装置**，会测量你的车辆通过时的速度。

- **超过 110 studs/s**：HUD 会闪示你的记录速度。你的个人最佳记录会保存到 **Speed Trap** leaderstat——试着打破记录吧！
- **超过 200 studs/s 且未开启紧急灯（ELS）**：你将被记录一项 **Speeding** 违规，可能触发 Polis Trafik 的执法。开启 ELS 的紧急队伍车辆例外。

**收费广场**——Plaza Tol Jersik 和 Plaza Tol Putra Mahkota——目前接受 CnG 扫描付款，带有 12-stud 的邻近提示。在此版本游戏中的收费**免费通过**（扫描会开闸但不会扣除 BR）。你的首次通过收费站会计入 *Bayar Tol* 发现挑战（+25 XP 一次性奖励）。驾驶座上带有 RFID 标签的车辆可以不扫描自动通过。

## 导航

使用 **GPS** 系统（可从顶栏 / 手机访问）为任何 POI 设置路径点。发光的光束会沿着道路引导你——普通玩家没有点对点传送。

游戏中存在 Fast Travel，但仅限于 **MYSverse 工作人员**（管理员 / 开发者），用于活动协调。如果你不是工作人员，你就得驾驶每一段旅程——所以请好好规划路线。你行驶的每一个 stud 都是 XP，所以风景路线通常是更有利的路线。
