# 房屋

Lebuhraya 拥有一套 **住宅系统**，你可以认领地块、放置房屋、用家具装饰它，并把它当作自己的基地使用。这是游戏中最具社交性的功能之一 — 邀请朋友过来、锁上门、举办一场甘榜式的聚会。

## 认领地块

地块位于 Bandar Seri Putra 的 **Pangsapuri Seri Putra**。可用地块总数由 server 决定；如果所有地块都被占用，你会看到 **0 available** 的提示。

1. 驾车前往 Pangsapuri Seri Putra。
2. 走到空的地块标记前。
3. 与认领提示互动。
4. 选择房屋模板（见下方）。

你的地块与账户绑定。离开 server 后你的房屋依然保留；回来时它仍在那里。

## 房屋模板

认领地块时你可以从四个模板中挑选：

- **Kampung_House_1** — 传统马来西亚甘榜风格的家
- **Kampung_House_2** — 另一种甘榜变体，布局不同
- **Kampung_House_3** — 第三种甘榜变体
- **Modern_Terrace_1** — 现代排屋 / 联排别墅风格

每个模板的室内面积相同，但外观特色各异。挑一个符合你角色扮演的模板。

## 自定义

房屋放置完成后，你可以：

### 油漆

更换房屋外墙的油漆颜色。

### 家具

从家具目录中添加物品。可用物件包括：

- **Bookshelf**
- **Single Bed**
- **Dining Table**
- **Coffee Table**
- **Chairs**
- **Mirror**

把物品放置在房屋内；移动和旋转它们，直到布局看起来合适为止。在当前版本的游戏中，放置家具 **不会扣除 BR** — 货币系统尚未上线（参见 [经济与 XP](/lebuhraya/economy#ringgit-br--tracked-but-not-yet-live)）。趁还免费时尽情享受装饰吧。

### 门

门可以 **上锁** 或 **解锁**。上锁的门会阻止其他玩家进入，包括朋友 — 想要有人陪伴时就解锁它。

### 灯光

切换室内灯光开或关。适合在夜间营造气氛。

## 谁可以进入？

- **你** — 任何时候都可以。
- **其他玩家** — 只有在你的门解锁时才能进入。
- **队友** — 默认没有特殊访问权限；适用相同规则。

## 离开或搬家

如果你想放弃当前地块以认领另一块，使用房屋菜单中的 **release** 选项。当你释放地块时，当前的房屋和家具都会被移除。释放前请考虑用截图保存你的布局设计 — 没有自动恢复功能。

## 关于价格与数量的说明

每间房屋和家具的确切 BR 费用，以及 server 端可用地块的总数，取决于实时游戏状态。游戏内菜单始终是这些数字的真实来源。
