# 常见问题与技巧

常见问题、提升游戏体验的小技巧,以及老玩家希望早点知道的事情。

## 常见问题

### 为什么我进入测试 server 后 XP 被重置了?

测试 server(包括 Studio 试玩)会故意将 XP 重置为 0,让开发者能在一个全新的账户上测试功能。"XP will be reset to 0" 的警告讯息是正常的。**你的正式游戏进度不会受影响** — 只有测试 server 是一次性的。

### 我打不开 vehicle spawner(MyPod)。为什么?

有几种可能:

- 你可能需要完全加载 — 加入后等几秒。
- 可能有 UI 遮罩盖在上面 — 关闭任何打开的菜单(Esc 或 Backspace)。
- 如果问题依旧,重新加入 server。

### 升级最快的方法是什么?

最快升级的公式:

1. **持续坐在 DriveSeat 上开车。** 被动 XP 是每 1500 stud 得 50 XP。
2. **做任务** — Food Delivery 在你达到 10 级后每次送餐给 250 XP。
3. **与同队朋友组 convoy** — 最高 +50% 加成。
4. **在长时间游玩前激活 2× XP token**。
5. **在 Double XP Weekend 期间游玩**(如果有公告)。
6. **订阅 Roblox Premium** 获得永久 +25%。
7. **购买 XP Boost gamepass** 获得永久 +50%。

将这些全部叠加起来,你的升级速度可以比独自一人开车的玩家快 10–20 倍。

### 我想当警察。怎么做?

警察队伍需要 gamepass:

- **KEMRonda** — 高速公路辅助(交通协助、拖车)。
- **Polis Trafik** — 交通法规执法。
- **Undercover Police** — 便衣执法。

购买你想要的队伍的 gamepass,然后从 Main Menu → Teams 标签加入。

如果你想在没有 gamepass 的情况下从事执法工作,**APJ**(300 级)是陆路交通局的队伍,但离达到还很遥远。

### 我怎么为我的车加油?

1. 开到加油站 — 高速公路上的 **PETROMAS**、Bandar Seri Putra 的 **Radtrol**。
2. 停在与你燃料类型相符的油泵旁(Petrol、Diesel 或 Electric)。
3. 屏幕上会出现提示 — 互动以加油/充电。
4. 每升扣除 BR。

燃料价格:Electric 1 BR/L、Petrol 2 BR/L、Diesel 3 BR/L。40 升满箱汽油需要 80 BR。

### 我的车没油了,困住了。现在怎么办?

你有三个选择:

1. **在 Jerrycan Shop 购买 JerryCan**,在任何加油站加满(匹配燃料类型),然后倒入你的油箱。
2. 从 MyPod 菜单 **despawn 再 respawn** 你的车辆。你会失去位置但获得满箱。
3. **请求 KEMRonda 玩家** 来拖车 — 他们可以把你带到最近的加油站。

### 我在哪里修车?

地图上放置了 **10 个维修区**。把你的车辆开进去,会出现提示 — 确认、保持停放,血量大约每 5 秒恢复 10%,直到完全修复。如果该区已被其他玩家使用,你无法进行维修。

Bandar Seri Putra 的 **Gravstone Workshop** 是 **改装店**(喷漆、车身),不是维修店 — 别去那里期望修车。

### 999 紧急系统怎么运作?

任何玩家都可以从他们的电话(topbar → 999 应用)拨打 999。值班的紧急响应人员(Bomba、Polis Trafik、Undercover Police、Kesihatan)可以在他们的 **Pager** 工具上看到事件并接受。

- 事件类型:**Car Crash**、**Injured Person**。
- 最短处理时间:接受后 **30 秒**(防止即时解决农取)。
- 同时最多可有 10 个事件开启。

完整流程请参阅 [Social Features → 999 Emergency](/lebuhraya/social-features#999-emergency-system)。

### 什么是伤害系统,为什么重要?

当 **Kesihatan(医护)玩家加入 server** 时,伤害系统 **会自动为所有人启动** — 包括你。这让你在每次获得 XP 时免费得到 **+15% XP 倍数**,与 Premium、convoy、token 和其他加成叠加。当最后一位医护离开时,它会再次关闭。

取舍:伤害系统开启时,严重碰撞可能对你造成需要医疗的伤害。这些伤害会变成医护响应的 999 事件 — 整个循环按设计运作。

**提示**:如果你在 server 大厅看到医护,你正在免费获得额外 XP。小心开车,你很少会需要治疗。

### 什么是 Passive 模式?

Passive 模式是一个开关,可以阻止基于工具的互动 — 武器、电击枪、绷带和类似装备 — 在 **双向** 上:

- 你不能对其他玩家使用工具。
- 其他玩家不能对你使用工具。

如果你不想被卷入战斗或紧急 roleplay,从 Main Menu 开启它。当你准备好参与时关闭它。注意:标准驾驶和任务动作不受影响 — passive 只封锁武器和医疗工具。

### 我可以拥有房子吗?

可以。到 Bandar Seri Putra 的 **Pangsapuri Seri Putra**,认领一块地,从四种房屋模板中选择(3 种 kampung 风格 + 1 种现代排屋),然后自定义。参阅 [Housing](/lebuhraya/housing)。

### 我怎么快速传送?

打开 **Fast Travel** 菜单(topbar 或主菜单)。选择目的地并确认。快速传送会 **暂停你从距离获得的 XP 累积 5 分钟**,以防止被利用 — 所以如果你想要最高 XP,别在长时间开车前快速传送。

### 我加入了 convoy 但没有获得加成。

检查这些:

- 你是否在另一位 convoy 成员的 **200 stud** 范围内?加成只在你们靠近时才适用。
- convoy 是否真的在运作?检查 convoy 面板 — 它会显示你目前的倍数。
- 你是 **乘客** 吗?乘客只获得驾驶者加成的一半。

### 什么是 token?

你物品栏中的消耗品,提供暂时性加成 — 2× XP、Season Pass XP 倍增器,或暂时性的队伍权限。从物品栏激活。参阅 [Social Features → Tokens](/lebuhraya/social-features#tokens-and-boosts)。

### 有单人/私人模式吗?

私人 server 可用(从 Roblox 体验页购买)。请注意 **私人 server 中经济系统被禁用** — 你不能赚取或失去 BR 和 XP。私人 server 最适合举办活动、roleplay 场景,或无压力地练习驾驶。

### 有促销代码吗?

有 — 打开 **Main Menu → Codes** 标签并输入代码。少数代码是永久有效的:

| 代码 | 奖励 | 使用次数 |

| `experience` | 250 XP | 最多 5 次 |
| `unlimitedpowarr` | 1,000 XP | 无限次 |
| `gokartracing` | 免费 Go-Kart(Special spawner) | 一次 |
| `fd2r` | 免费 Handai Civic FD2R(Special spawner) | 一次 |
| `renjer` | 免费 Renjer Loader 2012(Special spawner) | 一次 |
| `outlandish` | 免费 Masubi Outlander(Special spawner) | 一次 |
| `laksa` | 免费 ZulBezza 彩蛋 | 一次 |
| `MYS5KFOLLOWERS` | 免费 Intero Citie Bas Sekolah(Bus Driver spawner) | 一次 |

活动代码(限时)偶尔会在社区活动、大会和道歉补偿时出现 — 关注 MYSverse 频道以获取。

## wiki 可能没提到的技巧

### "长时间开车"总是有利可图

无论你在做什么,坐在 DriveSeat 上注意路况都能让你免费获得 XP。即使在前往任务的路上,你也在赚。

### 让你的车辆与任务匹配

Tayoti Hiace 是载客厢型车 — 别把它开上 Race Track。Perosatu Myvi 适合 food delivery 但对货运没用。查看 [Vehicles](/lebuhraya/vehicles#vehicle-categories) 中的车辆分类。

### 电动车的优势

电动车每升 1 BR,比柴油每升 3 BR 便宜。在数百小时的驾驶中,这会累积成相当大的差异。能用的时候试试 Besla Y、MYD Atto III、Ionic I/V 或 CITO Ami。

### 刷之前先结伴

招募一两个同队的朋友。开始 convoy。在 200 stud 范围内一起行动。你每小时的 XP 相比独自玩大约翻倍。

### 把 token 留给专注时段

别开了 2× XP token 然后就分心 — 计时器不会暂停。在你有一小时不间断游玩计划时再使用它们。

### 留意 Double XP Weekend

密切关注 MYSverse 社区频道。Double XP Weekend 是使用其他加成最划算的时机。

### 如果商店不显示价格

价格因商店而异,有时因商品而异。购买前总是检查游戏内菜单,特别是房屋家具等昂贵物品。

### 尊重 roleplay

Lebuhraya 是一款 roleplay 游戏。认真上 999 医疗班的玩家不喜欢被撞出路;试图抓超速者的交通警察不喜欢被嘲笑。带着尊重开车;社交场景是乐趣的一半。

### 回报 bug 和反馈

如果你遇到 bug 或有建议,游戏内的 **Report** 系统(MYS_ReportSystem ScreenGui)会直接发送给 MYSverse 团队。使用它 — 开发者会阅读这些。
