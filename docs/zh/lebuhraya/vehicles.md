# 车辆

Lebuhraya 配备了 **127 款车型** — 紧凑型轿车、SUV、摩托车、卡车、巴士,以及一长串紧急/服务变种车款。本页介绍如何生成车辆、保持燃料充足、撞车后该怎么办,以及车库内有哪些车款的概览。

## 生成车辆

打开 **MyPod** 菜单(车辆生成器)。

- **桌面端**:点击顶栏的 MyPod 图标
- **移动端**:点击顶部聚合区的图标

浏览列表,挑选一辆车,然后点击 **Spawn**。车辆会出现在您面前,并已注册在您名下。要驾驶时,走近并按下 **F**(键盘)、**X**(手柄),或点击 **Drive**(移动端)。

### 所有权

- 当您生成车辆时,您即成为其车主。您的名字会出现在驾驶座上。
- 其他玩家**无法进入您的驾驶座**,除非您解锁。
- 乘客始终可以进入乘客座位。
- 当您离开 server 时,您的车辆会消失。
- 您可以从 MyPod 菜单中销毁自己的车辆。

## 1 级时我可以开什么车?

您从一开始就可以使用 **27 辆车** — 足以找到您喜欢的车款。最便宜的是 **Handai EX5** 摩托车,售价 BR 1050。Premium / 超级跑车的入门车款包括 NSN GT-R(BR 3,750)、Bagaini Chiron(BR 3,000)和 McLari 750(BR 3,000)— 比现实中便宜,但大多数属于"Special"类别(无法升级)。较常见的 1 级民用车价格约在 BR 1,800–6,000 之间。

## 价格与解锁

车辆价格大致随等级而变化。粗略指南:

| 等级 | 典型价格区间 |
|---|---|
| L1–10 | BR 1,050 – 8,000 |
| L20–40 | BR 10,000 – 22,400 |
| L50–100 | BR 15,000 – 20,000 |
| L125–150+ | BR 17,000 – 25,000(Prem/超级跑车区间) |

部分仅限团队的车辆没有公开价格 — 它们会随着您在相应职业分支中的晋升而解锁。**117 辆可生成车辆中约有 55 辆**为职业锁定(您需要在合适的等级加入合适的团队)。

约有 **28 辆车**通过车辆生成器中的 **Dealership** 视图出售 — 这些是从 Myvi 到 Mustang 的标准民用选择。

## 燃料

每辆车使用三种燃料中的其中一种。

| 燃料 | 每升价格 |
|---|---|
| **Electric** | 1 BR |
| **Petrol** | 2 BR |
| **Diesel** | 3 BR |

默认油箱容量为 **40 升**。一个 JerryCan 装载 10 升。

### 加油

将车开到加油泵旁,屏幕上会出现 "Pump" 或 "Charge" 提示。提示仅会针对您车辆所使用的燃料类型出现 — 如果您把燃油车开到柴油泵旁,游戏会提示您 "this is the wrong type of pump, find a Petrol Pump"。JerryCan 也是同理 — 在靠近油泵之前请选择正确的燃料类型。

### 加油地点

- **PETROMAS Petrol Station** — 高速公路上的北行和南行加油站
- **Radtrol Petrol Station** — 位于 Bandar Seri Putra
- **Kompleks Petroliam dan Gas Radtrol** — 位于 Jersik 附近的大型加油综合站

### 油用完了

如果油箱归零,引擎将停止运转。选项:

1. 在 **Jerrycan Shop** 购买 JerryCan,在加油站加满,然后手动加油。
2. 销毁车辆并从 MyPod 生成新的车辆。
3. 请 KEMRonda 玩家拖车。

## 撞车与维修

车辆会因碰撞而受损。受损严重的车辆会开始冒烟、性能下降,或完全停止。

### 您的选项

- **驶入维修区**。地图上散布着 **10 个维修区** — 寻找绿色的维修指示牌。在区域内停车会逐渐恢复您的车辆(每 5 秒大约 10% 生命值)。当您进入时屏幕上会出现确认提示。
- **呼叫拖车**。拥有拖车的 KEMRonda 玩家可以将您拖到工坊。
- **销毁后重新生成**。从 MyPod 菜单中,您可以销毁损坏的车辆并生成新的车辆。如果您赶时间,这是最快的选项。

如果您是新手且觉得撞车令人却步,首次发生时您将看到 **Crash Tutorial** — 这就是您游戏内的入门指南。请小心驾驶,使用手刹(**Space**),并对马来西亚高速公路速度持保留态度。

## 在 Gravstone Workshop 进行定制

位于 Bandar Seri Putra 的 **Gravstone Workshop** 是游戏的喷漆与视觉定制商店 — **不是**维修中心。将您的车驶入工坊的内部区域,工坊 GUI 会出现,并提供喷漆和车身选项。

- **进入需要 Workshop gamepass。**如果您没有,当您打开喷漆选项时游戏会提示您购买。
- 您可以重新涂色的**喷漆区**包括车身、车窗、轮胎和轮圈 — 每个区域可以选择不同的颜色。
- **颜色选取器**:拖动色调、饱和度和明度;调整时车辆会实时更新。点击 Confirm 应用;点击 Cancel 取消。

此处保存的喷漆将应用于您当前生成的车辆,持续整个会话期间。可解锁的喷漆(chrome、gradient、matte — 见 *Passes and Boosts*)需通过另一个菜单应用。

## 定制

车辆定制取决于具体型号。常见选项包括:

- 喷漆颜色
- 车牌号码
- 车身配件

请查阅游戏内车辆的菜单(部分车辆有定制面板,部分则没有)。团队配发车辆(警察、Bomba)通常被锁定为团队涂装。

## 紧急灯(ELS)

所有紧急团队车辆(Bomba、Polis Trafik、Undercover Police、Kesihatan、KEMRonda)都装有使用游戏 ELS(Emergency Lighting System)的灯条。切换 ELS:

- 播放 **三种闪烁模式**之一 — 快速交替、慢速交替和快速复杂。
- 复制到 server 中的所有玩家,这样每个人都能看到您的灯光。
- 授予隐式的测速**豁免** — 如果您在 ELS 启用时超过 200 stud/s,您不会被记录超速违规(紧急响应进行中)。请勿滥用;反作弊系统仍在监视。

仅在响应事件时使用 ELS。开着灯巡游属于社区礼仪上的禁忌。

## 高级车辆控制

六辆专用车辆配有 **拖车/绞盘面板** — 这是一个适合移动端的横屏 HUD,当您与车厢内的控制面板互动时会出现:

| 车辆 | 团队 | 面板 |
|---|---|---|
| iZu Tow Truck KEMRonda | KEMRonda | Tow Boom Controls |
| iZu Tow Bed KEMRonda | KEMRonda | Flatbed Controls |
| Tayoti Hilux KKN RRV | Kesihatan | Rescue Winch Controls |
| Merze Atego LFRT Bomba | Bomba | Ladder Winch Controls |
| Sanica P310 FRT Bomba | Bomba | Ladder Winch Controls |
| Sanica 94D LFRT Bomba | Bomba | Ladder Winch Controls |

**键盘快捷键**(面板打开后):

| 键 | 操作 |
|---|---|
| **1** | 切换 Winch 1 的 Attach 模式(点击车辆/部件以将其设为目标) |
| **2** | 切换 Winch 2 的 Attach 模式(配备第二个绞盘的平板拖车/拖车) |
| **X** | 取消 attach 模式 |
| **Z** | Weld(将附加的车辆与您的车辆熔接) |
| **V** | 释放 / 取消熔接 |
| **[** | 降下平台 / 坡道 |
| **]** | 升起平台 / 坡道 |

**访问规则**:您始终可以使用面板,前提是 (a) 您拥有该车辆,(b) 您在对应的团队(KEMRonda / Bomba / Kesihatan)中,或 (c) 您是管理员。否则会出现限速的 "Access Denied" 横幅。

控制方式在六辆车之间共享,所以一旦您学会一辆,您就学会了全部。
