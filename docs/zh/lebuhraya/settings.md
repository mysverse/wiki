# Lebuhraya 设置与自定义

Lebuhraya 提供多种设置选项，帮助你自定义游戏体验、优化性能并控制交互方式。

## 访问设置

- 大多数常规设置可在**主菜单**中找到。
- 部分特定开关或选项可能通过游戏画面上的专用界面按钮访问。

## 性能设置

这些设置可以帮助提升性能，尤其适用于配置较低的设备。

- ### 全局阴影开关

  - **推出版本：** v3.3.25
  - **功能：** 允许你切换整个游戏的阴影渲染。关闭阴影可以显著提升性能。
  - **访问方式：** 主菜单。

- ### 树木渲染开关

  - **推出版本：** v3.3.15
  - **功能：** 允许玩家关闭树木的渲染。这有助于减少视觉杂乱并提升性能。树木也会被加载到内存中（而非流式加载），以潜在地减少崩溃。
  - **访问方式：** 主菜单右上角的按钮。

- ### 光照技术
  - **变更：** 在 v3.4.21 中，光照技术从 "Future" 更改为 "ShadowMap"，以提升低端设备的性能。这可能是由开发者管理的全局设置。

## 游戏与操控设置

- ### 触屏冲刺功能

  - **推出版本：** v3.3.26
  - **功能：** 为触屏设备启用冲刺切换功能。键盘用户可使用左 Shift 键。
  - **访问方式：** 触屏友好的**设置菜单**。

- ### 首选出生地点

  - **推出版本：** v3.0.11
  - **功能：** Malaysian 和 Visitor 团队的玩家可以将首选出生地点设置为 Putra Mahkota 或 Jersik。
  - **访问方式：** 主菜单。
  - **更新：** 在 v3.5.8 中，这些团队的默认出生点统一设置为 Putra Mahkota，但切换功能可能仍然可用。

- ### MYS 模块相机行为自定义

  - **推出版本：** v3.0.13
  - **功能：** 玩家可以禁用 MYS 模块的默认相机行为（此行为的具体细节未说明，但可能指自定义相机脚本）。

- ### 车辆碰撞开关
  - **位置：** 车内屏幕上的按钮。
  - **功能：** 允许切换车辆碰撞。
  - **视觉变更（v3.0.13）：** 碰撞按钮的"开启"状态颜色从绿色更改为默认黑色。
  - **移动端界面（v3.0.6）：** 调整了碰撞按钮的大小，以便在移动端驾驶界面上更容易操作。

## 界面与 UI

- **新 UI 字体（v3.4.4）：** 游戏采用了 Roblox 的 "Builder Sans" 字体作为文本 UI 元素，以提高清晰度和可读性。
- **主菜单访问（v3.5.9）：** 现在可以通过顶部栏访问主菜单，无需重置角色。
- **移动端驾驶界面（v3.0.6）：** 通过调整部分按钮大小并移除驾驶时左下角的设置按钮来精简界面，提供更清爽的视野。
- **手柄支持（v3.3.27）：** 持续改进中，包括修复手柄无法选择主菜单播放按钮的问题。

## 音频设置

- **MyPod 个人音乐播放器（v3.0.11）：** 通过顶部栏的音乐图标访问。可以收听仅自己能听到的精选音乐。
- **警笛音量（v3.4.3）：** 降低了 Kesihatan 救护车的警笛音量。这表明部分音频水平由开发者根据反馈进行全局管理。

## 开发者与管理员相关

- **头顶等级 GUI（v3.0.3）：** 显示玩家的显示名称而非用户名。同时显示玩家等级。
- **管理员聊天可见性（v3.3.4）：** Adonis 管理员在聊天中看到完整的 Roblox 用户名而非显示名称，以便于管理。

这些设置为你的 Lebuhraya 体验提供了一定程度的控制。随着游戏更新，可能会有新的选项推出。请经常查看主菜单和屏幕上的 UI 元素以获取最新设置。
