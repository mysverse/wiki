# 操作

Lebuhraya 支持键盘、手柄、触控(手机/平板)和 VR。游戏会自动侦测你的平台并在游戏内显示正确的提示 — 如果屏幕上显示某个操作,就用那个。以下参考是常见按键配置的样子。

## 进出车辆

站在距离驾驶座或乘客座约 **6 stud** 以内。

| 平台 | 按键 |
|---|---|
| 键盘 | **F** |
| 手柄 | **X**(手柄面键) |
| 触控 | 点击屏幕上的 **Drive** / **Sit** 按钮 |
| VR | **Right trigger (R2)** |

同样的输入也用于离开座位。

### 座位权限

驾驶座通常锁定给车辆拥有者。如果你尝试坐进别人的车,会被拦下,除非对方为你解锁。

## 驾驶

驾驶使用 A-Chassis 系统。默认键位:

| 动作 | 键盘 | 手柄 |
|---|---|---|
| 加速 | **W** | Right trigger (RT) |
| 刹车 / 倒车 | **S** | Left trigger (LT) |
| 向左转 | **A** | Left stick |
| 向右转 | **D** | Left stick |
| 手刹 | **Space** | A / RB |
| 切换信号灯 / 车灯 | **E**、**Q**、**I**、**J**、**H**(不一) | 参考游戏内提示 |
| 喇叭 | **H**(大多数车辆) | Right stick click |

> 游戏中较旧与较新的 chassis 系统之间,车辆灯光的操作方式有所不同。请始终依照你所乘坐车辆在屏幕上显示的提示操作。

## 步行

标准 Roblox 操作。

| 动作 | 键盘 | 手柄 | 移动端 |
|---|---|---|---|
| 移动 | **W / A / S / D** | Left stick | 屏幕上的方向键 |
| 跳跃 | **Space** | A | 跳跃按钮 |
| 冲刺 | (游戏内启用 shift-to-run) | Left stick click | 冲刺切换 |
| 镜头 | 鼠标 | Right stick | 滑动 |

## 界面

| 动作 | 键盘 | 移动端 |
|---|---|---|
| 聊天 | **/**(斜线) | 点击聊天图标 |
| 主菜单 | 顶栏图标 | 顶栏图标 |
| 手机 | 顶栏图标 | 顶栏图标 |
| 平板 | 顶栏图标(如果可用) | 顶栏图标 |
| 设置 | 主菜单中的齿轮 | 主菜单中的齿轮 |
| 关闭任何弹窗 | **Esc** 或 **Backspace** | 点击关闭 |

## 工具(工作装备)

当你手持工具 — 斧头、雷达测速枪、电击枪、呼机、消防水带 — 时,操作方式是该工具专属的。请参考屏幕上的提示,或查阅使用它的团队的工作页面。

## 移动端提示

- 游戏适配移动端布局;在手机上开车可行,但在平板上更方便。
- 大多数 ProximityPrompt(进门、与 NPC 交谈、开始工作)会在屏幕上显示一个圆形按钮。点击即可。
- 驾驶时方向盘显示在 HUD 的左侧。

## 辅助功能

- Roblox 系统级设置同样适用 — 色盲滤镜、UI 缩放、文字转语音。
- 可以切换被动模式来阻止工具和武器的互动(你的 `Passive` 玩家属性)。如果你不想被卷入战斗或紧急角色扮演,这很有用。
