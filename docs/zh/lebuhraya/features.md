# Lebuhraya 游戏功能

Lebuhraya 不仅仅是驾驶；它还包含了丰富的功能，打造出引人入胜的角色扮演和成长体验。以下是一些核心游戏系统：

## 燃油系统（Beta 版）

- **推出版本：** v3.5.0（"Fuel Up, Rev Up"）
- **机制：**
  - 车辆会消耗燃料，需要在地图上 5 个加油站中的任意一个进行加油。
  - 燃油耗尽时车辆将无法运行，可能需要 KEMRonda 提供拖车服务。
  - 目前加油**不消耗**任何游戏内货币或 XP。
  - 燃油类型和消耗值会根据反馈进行调整。
- **更新：**
  - 在每辆车的驱动脚本中进行了内部重新实现（v3.5.2）。
  - 根据反馈调整了燃油消耗数值（v3.5.2）。
  - 修复了触屏/移动设备上不显示油量表的问题（v3.5.13）。

## 法律与违规系统

- **推出版本：** v3.5.12
- **被动锁定：**
  - 当玩家违反游戏内法律时触发（例如被雷达测速枪或测速陷阱抓到超速）。
  - 被动锁定激活期间，无法切换被动模式。
  - 技术高超的驾驶员仍可通过灵活操作甩掉警察。
  - 未来可能扩展功能，如逆行驾驶、银行抢劫等。
- **积分制违规与逮捕系统：**
  - 违规行为会产生积分；积分会累积直到达到阈值。
  - **阈值：** 10 分 → 玩家可被逮捕。
  - **示例：** 超速 = 4 分（被抓 3 次且速度 ≥200 SPS 即可被逮捕）。
  - 被逮捕的玩家除重生外没有其他直接惩罚。
  - 逮捕范围 64 格内的所有执法 (LE) 人员可获得 **100 XP**。
- **参与的执法 (LE) 团队：** Polis Trafik（游戏通行证）、KEMRonda（游戏通行证）、APJ（免费）。
- **平衡性：** 积分值、速度阈值和逮捕规则会根据反馈进行调整。

## 钓鱼系统

- **推出版本：** v3.5.5
- **地点：** Bandar Seri Putra 附近的旧景观路（_jalan lama_）。
- **钓鱼方法：**
  1. 前往**渔港**并装备你的**钓鱼竿**。
  2. 点击/点按水面抛出鱼线。
  3. 当出现"晃动"圆圈时，快速点击/点按它们。
  4. 长按点击/触摸，将白色矩形保持在移动的"钓鱼"进度条内（保持绿色区域）。
  5. 你的渔获将出现在物品栏中。
- **出售渔获（换取 XP）：**
  1. 接近**鱼贩**。
  2. 装备钓鱼竿后，点击/点按物品栏中的鱼进行选择。
  3. 按下绿色的**出售**按钮。
- **出售倍率（XP）：**
  - **渔港：** 0.5 倍 XP
  - **Bandar Seri Putra：** 1 倍 XP
  - **Kampung Merbang：** 2 倍 XP
- **更新：**
  - 修复了排行榜渲染和钓鱼竿目标问题（v3.5.6）。
  - 捕获后新增 2 秒冷却时间（v3.5.6）。
  - 为钓鱼竿工具添加了 30 秒故障保护机制（v3.5.12）。

- **地点：** Bandar Seri Putra 的 SMK Lebuhraya（靠近 Putra Mahkota 出生点、Masjid BSP）。
- **机制（v3.2.0 — 初始版本，细节可能有所变化）：**

  - 学校测验排行榜（持久化服务器和玩家列表）（v3.3.13）。答题越快，分数越高。

## XP、进度与挑战

- **等级系统：** 最高等级为 **500**（v3.4.23）。每个等级需要 1,000 XP（v3.0.5）。
- **获取 XP：**
  - 驾驶：每 1,000 格距离 +50 XP（v3.0.5，格距追踪从 2000 格降低）。
  - 访问地标：每个会话中每个独特地标 +100 XP（v3.0.3）。
  - 团队专属任务（详见[团队介绍](/zh/lebuhraya/teams)）。
  - 学校测验：每答对一题 +250 XP（v3.3.13）。
  - 挑战。
- **XP 倍率（v3.3.24）：**
  - **50% XP Boost 游戏通行证：** 1.5 倍 XP。
  - **Roblox Premium：** 1.25 倍 XP。
  - 两者叠加最高 1.75 倍 XP。
  - 倍率不与特殊活动叠加（例如双倍 XP 周末上限为 2 倍）。
- **每日与每周挑战（Beta 版，v3.3.24）：**
  - 每天马来西亚时间（GMT+8）凌晨 12 点重置，每周一重置。
  - 多种挑战类型，包括职业专属挑战和独特任务。
  - 奖励可包括 XP、装饰品和车辆。
- **挑战可解锁车辆：**
  - Tayoti Supra Mark V（Jersik Race Track 单圈 < 45 秒）（v3.4.4）。
  - NSN S15 Silvia（Jersik Race Track 3 圈，每圈 < 45 秒）（v3.4.19）。
  - NSN Nismo（Quiz Master Advanced：在 SMK MYS II 中 28 道题每题 < 2 秒）（v3.5.0）。
- **等级跳过 / 级别跳过（v3.3.25）：**
  - 可购买的跳过包，每个包含 **1,000 XP**（相当于 1 级）。
  - 批量套餐最多 50 级（最高可享 30% 折扣）。

- **新景观路线（Jalan Lama）（v3.3.12）：** 从 Bandar Seri Putra 开始，在鬼屋附近结束。可绕过 Putra Mahkota 收费站。

- **Roblox 地形升级（v3.3.16）：** 从 Kampung Merbang 开始，营造更传统的氛围。

  - Putra Mahkota 新可玩区域建设准备（v3.4.24）。
  - 石油天然气工业园区建设用地（v3.5.10）。
- **出生点切换器图片（v3.4.9）：** 显示传送目的地的背景图片。

## 房屋定制

- **地点：** Kampung Merbang（v3.0.0）。
- **机制：** 在地块上建造房屋。修复了建造问题（v3.1.4）。修复了 Kampung Merbang 房屋地块问题（v3.4.1）。
- **家具：** 甘榜风格家具（v3.0.1）。改进了家具放置碰撞逻辑（v3.4.11）。

## 伐木系统

- **地点：** Jersik 区域的 H Plantation。
- **运作方式：**
  1. 加入**伐木工**团队（150 级免费解锁）。
  2. 从工具中装备你的**斧头**。
  3. 接近一棵树并挥斧砍伐。
  4. 收集木材。
  5. 在**木材商店**出售木材以获取 XP 奖励。
- 树木被砍伐后会重新生长，允许持续采伐。

## 货运与物流系统

- **团队：** 物流运输（75 级免费解锁）。
- **运作方式：**
  1. 从**卡车生成器**生成一辆卡车。
  2. 在指定的**挂钩点**连接拖车。
  3. 按照路线指引在各地点之间运送货物。
  4. 完成运送即可获得 XP 奖励。
- **工具：** 物流指南，用于追踪配送进度。

## 废物处理系统

- **团队：** 废物处理（25 级免费解锁）。
- **运作方式：**
  1. 从**废物车生成器**生成一辆垃圾车。
  2. 驾驶到 Lebuhraya 各处的垃圾桶收集垃圾。
  3. 将收集的垃圾带到**焚化炉**进行处理。
  4. 完成任务即可获得 XP。
- **工具：** 废物处理指南，用于追踪收集点。

## 医疗系统（KKN Medic）

- **团队：** Kesihatan（200 级免费解锁）。
- **医院区域：** Bandar Seri Putra 的 KKN Medic 医院。
- **工具：** 夹板、急救箱、绷带、Kesihatan 手册。
- **玩法：**
  - 通过值班切换按钮切换上班/下班状态。
  - 响应受伤玩家并提供医疗救助。
  - 使用操作菜单执行医疗操作。

## 受伤系统

- **机制：**
  - 玩家可能因车辆碰撞和其他事故而受伤。
  - 受伤时屏幕上会出现伤势通知。
  - Kesihatan 团队成员可以响应并治疗受伤玩家。
  - 碰撞服务会生成可导致伤害的碰撞事件。

## KeluargaMart（便利店）

- **地点：** Lebuhraya 各处的多个位置。
- **功能：**
  - 浏览并购买商店库存中的物品。
  - 物品包括食物、饮料和其他消耗品。
  - 在商店附近时通过 KeluargaMart 界面进行交互。

## 代币系统

- **机制：**
  - 代币是提供临时增益或效果的特殊物品。
  - 激活的代币会显示在屏幕上的**激活代币 HUD** 中。
  - 代币可通过挑战、活动获得，或从商店购买。

## 交通系统

- **交通信号灯：** 自动交通灯控制器管理各个路口，配有可配置的绿灯、黄灯和红灯阶段。
- **闯红灯检测：** 驾驶时闯红灯将触发**闯红灯违规**（1分）和60秒被动锁定。10秒冷却时间防止重复处罚。
- **测速陷阱：** 自动测速检测区域，抓捕超速车辆。

## 其他系统

- **R15 角色（v3.3.16）：** 游戏过渡到现代 R15 角色骨架。
- **节日装饰：** 斋月（v3.3.0）、开斋节（v3.3.15）、农历新年（v3.4.30）、屠妖节（v3.4.22）、新年倒计时（v3.4.26）。
- **数据管理：** DataStore2 模块已替换为 Roblox 的迁移工具（v3.5.9）。如有数据丢失请上报。
- **私人服务器：** 私人服务器中 XP 不保存（已添加通知，v3.3.23）。

以上功能共同构建了 Lebuhraya 的动态演变世界。请关注游戏内公告以了解最新内容！
