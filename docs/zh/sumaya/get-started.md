# 入门

![Sumaya](/images/sumaya/sumaya_feature.webp)

欢迎来到 **Sumaya**！本指南将帮助你在这个马来西亚主题的开放世界中迈出第一步——探索充满活力的城市、从事不同的职业、赚取 Sumaya Ringgit，并建立属于自己的生活。

::: tip 欢迎！
Sumaya 充满了各种职业、活动和值得探索的事物。快来加入吧！
:::

## 主菜单导航

<!-- TODO: 添加主菜单截图 -->
<!-- ![主菜单](/images/sumaya/get-started/main-menu.png) -->

首次加入 Sumaya 时，你会看到**主菜单**，包含以下选项：

- **开始游戏：** 直接进入城市。
- **队伍：** 在访客、马来西亚公民、POLIS、BOMBA 和 Kesihatan 等角色之间切换。
- **职业：** 查看你的职业进度和当前角色信息。
- **商店：** 购买游戏通行证和其他物品。
- **更新：** 查看最新的游戏更新和公告。

主菜单还有一个观景相机，展示 Sumaya 的地标，包括清真寺、钟楼、火车站、海滩和街道。

## 赚取 Sumaya Ringgit（SR）

<!-- TODO: 添加钱包或ATM截图 -->
<!-- ![钱包](/images/sumaya/get-started/wallet.png) -->

Sumaya 中的一切都使用 **Sumaya Ringgit（SR）**。你初始拥有**钱包中 SR 50** 和**银行账户中 SR 1,000**。

通过以下方式赚取更多 SR：

- 完成工作（快餐、钓鱼等）
- 从事团队职业（POLIS、BOMBA、Kesihatan）
- 钓鱼并出售渔获

### 管理你的资金

- **钱包：** 从工具栏访问钱包工具以查看手头现金。你还可以使用 Sadaqah 功能向其他玩家捐赠 SR。
- **ATM：** 访问城市中五台 ATM 机器中的任何一台来存款、取款或在账户之间转账。

## 初始工具

每个玩家都会在库存中获得以下工具：

- **手机：** 通过游戏内智能手机访问应用和功能。
- **钱包：** 查看手头现金并向其他玩家捐赠。
- **手电筒：** 在夜间照亮黑暗区域。
- **鞭炮：** 燃放节日鞭炮。
- **芝士汉堡：** 随时食用以恢复饥饿值。

## 探索城市

<!-- TODO: 添加城市全景或地标截图 -->
<!-- ![城市概览](/images/sumaya/get-started/city-overview.png) -->

Sumaya 拥有一座详细的马来西亚主题城市，有很多值得探索的地方：

- **清真寺：** 城市中心的著名地标。
- **钟楼：** 远处可见的壮观塔楼。
- **火车站：** 城市中的交通枢纽。
- **海滩：** 适合放松和钓鱼的滨水区域。
- **医院：** 配备病床、MRI 和 X 光机器的完善医疗设施。
- **中华大会堂和建筑：** 反映马来西亚多元文化遗产的华人社区建筑。
- **商铺和店铺：** 各种商业区域，设有商店和摊位。
- **排屋：** 住宅区，你可以在这里认领地块并建造房屋。
- **变电站：** 分布在城市各处的公用设施基础设施。
- **路灯：** 夜间照亮道路的灯柱。

城市还设有动态的**昼夜循环**和带有雨效的**天气系统**，为世界增添生机。

## 基本操控

- **移动：** 标准 Roblox 控制（WASD / 摇杆）。
- **工具：** 从屏幕底部的库存工具栏中选择工具。
- **互动：** 走近物体并使用出现的接近提示进行互动。
- **手机：** 点击手机工具以打开游戏内智能手机。
- **钱包：** 点击钱包工具以查看现金余额。

## 早期必做事项

1. **观看教程：** 欢迎教程涵盖所有基础知识——不要跳过！
2. **探索城市：** 参观清真寺、钟楼、海滩和其他地标。
3. **尝试钓鱼：** 前往钓鱼区赚取你的第一笔 SR，了解马来西亚鱼类。
4. **找一份工作：** 在快餐店工作以获得稳定收入。
5. **租一辆车：** 访问经销商，租一辆 Protan Wira 以更快地出行。
6. **认领房屋：** 找到一块住宅地块，建造你的第一个家。
