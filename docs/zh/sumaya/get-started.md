# 入门

你在 Sumaya 的头几分钟。本页介绍主菜单、队伍选择、操作,以及你进入游戏世界后应做的第一件事。

## 主菜单

当你加入 Sumaya 时,你会在出生前看到一个主菜单屏幕。在这里你选择一个**队伍**,它决定你扮演的角色 - 普通市民、警察、消防员或医护人员。你选择的队伍会影响你开始时拥有的工具和能从事的工作。

你之后可以更换队伍。第一次的选择不是永久的。随便选一个你觉得有趣的就好。

当你准备好后,按下 **Jom**(马来语中的 "let's go")进入游戏世界。

## 选择队伍

共有五个队伍。其中三个(POLIS、BOMBA、Kesihatan)是有特定职责的紧急服务队伍。另外两个(Visitors 和 Malaysians)让你以普通人身份游玩。

| 队伍 | 他们的职责 | 任何人都可加入? |
|------|--------------|------------------|
| Visitors | 普通市民,默认选项 | 是 |
| Malaysians | 普通市民,为已验证小组成员设的装饰性队伍 | 仅限 Rank 3+ 的 MYSverse 小组成员 |
| POLIS | 警察:逮捕罪犯、巡逻、执行限速 | 是 |
| BOMBA | 消防队:扑灭火灾、执行救援 | 是 |
| Kesihatan | 医疗:治疗伤患、运营医院 | 是 |

详见[队伍](/sumaya/teams)了解每个队伍的更多信息,包括你开始时拥有的工具。

**不确定该选哪个?** 从 **Visitors** 开始。你可以立即尝试民间工作(钓鱼、mamak、棕榈油),之后在你逛过一圈后可以切换到紧急服务队伍。

## 你的起始资金

每位新玩家开始时拥有:

- 钱包里的 **100 Ringgit**
- 银行里的 **1000 Ringgit**

这足以租用几辆民用车辆或购买一栋基础的 tier-1 房屋。详见[经济](/sumaya/economy)了解更多关于金钱的信息。

## 操作

Sumaya 使用标准的 Roblox 移动操作。在 PC 上:

| 动作 | 按键 |
|--------|-----|
| 移动 | W、A、S、D |
| 冲刺 | Left Shift(按住) |
| 跳跃 | Space |
| 与提示互动 | E(当出现时) |
| 打开手机 | 屏幕上的手机按钮,或点击图标 |
| 聊天 | /(斜杠)聚焦聊天栏 |

在移动设备和游戏手柄上,屏幕上的按钮和标准的 Roblox 游戏手柄绑定按预期工作。

## 你的头五分钟

以下是你出生后立即要做的事:

### 1. 打开你的手机

手机是你的主要中心。点击屏幕上的手机图标打开它。你会看到关于职业、任务、紧急呼叫、设置等应用。详见[手机和 UI](/sumaya/phone)了解完整列表。

打开手机也算作你第一个任务的一个步骤。

### 2. 查看任务应用

你会自动开始 **Welcome to Sumaya** 任务链:12 个任务,教你基础并在你完成时支付 XP 和 Ringgit。每个任务都会准确告诉你下一步要做什么。

打开手机上的 **Quest** 应用查看你当前的目标。红色的引导光束和小地图上的标记通常会指引你到正确的地方。详见[任务](/sumaya/quests)了解完整任务链。

### 3. 生成一辆车

走到任何**车辆生成亭**前(留意散布在城里的提示)。选一辆车,租或买,然后开走。大多数入门级民用车辆是免费生成的 - 更高级的需要 Ringgit。

详见[车辆](/sumaya/vehicles)了解完整列表以及生成器的运作方式。

### 4. 开始一份民间工作

走到任何**职业站**前(钓鱼码头、mamak 摊、棕榈种植园),按下互动提示来打卡。你工作时会赚取 Ringgit 和 XP。工作在[职业概览](/sumaya/jobs)中介绍。

如果你选择了紧急服务队伍(POLIS、BOMBA、Kesihatan),你不需要打卡 - 你一出生就上岗。

### 5. 探索

步行、开车或飞行。有地图、地标、商店、医院、警察局、消防局、海滩,以及众多社区。小地图(屏幕右上角)显示你周围的情况。

## HUD 概览

当你在游戏世界中时,你会看到:

- **小地图**(右上角):显示你的位置、附近的道路和建筑物、路径点,以及当任务指向某处时的红色引导光束。点击展开。
- **状态小工具**(右上角,小地图下方):你的**市民等级**和进度条,加上显示你活动领域的第二行(紧急服务的等级,民间班次的等级 / 上限)。
- **任务卡**(左上角):显示你当前的目标(活动任务步骤、当前火灾、你正在治疗的病人等)。如果挡路,点击 V 形按钮折叠它。
- **TopBar 图标**(顶部中央):小型的 Map / Tasks / Status / Open Menu 图标。点击任何一个来隐藏该 HUD;点击 **Open Menu**(房屋符号)在游戏中重新打开主菜单而不死亡。
- **钱包 / 银行余额**:在屏幕顶部附近可见。

## 冲刺

按住 **Left Shift** 或 **Left Control** 来冲刺。步行速度为 10 studs/s;冲刺将其加倍至 20。当前 build 中没有耐力门槛 - 你想冲刺多久就冲刺多久。(游戏代码中有一套完整的饥饿/耐力/卫生系统,但负责为每个角色初始化这些数据的 server 处理程序目前已被禁用,因此实际游戏中没有一个会运行。)

当你进入车辆座位时,冲刺会自动关闭。

## 接下来呢?

- 选择一个[队伍](/sumaya/teams)和一个[职业](/sumaya/jobs)。
- 阅读[手机和 UI](/sumaya/phone)以便了解东西在哪里。
- 查看[任务链](/sumaya/quests)获得有结构的攻略。
- 当你准备好赚真正的钱时,看看[经济](/sumaya/economy)了解收入最高的工作在哪里。
