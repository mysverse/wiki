# 入门

你在 Sumaya 的头几分钟。本页介绍主菜单、队伍选择、操作,以及你进入世界后应该做的第一件事。

![Sumaya](/images/sumaya/sumaya_feature.webp)

## 主菜单

当你加入 Sumaya 时,你会在出生前看到一个主菜单画面。从这里你选择一个**队伍**,它决定你扮演什么角色 - 普通平民、警察、消防员或医护人员。你选择的队伍会影响你初始拥有的工具和你能做的工作。

你之后可以更换队伍。第一次的选择没有任何永久性。随便挑个听起来好玩的就行。

准备好后,按 **Jom**(马来语中的 "let's go")进入世界。

## 选择队伍

共有五个队伍。其中三个(POLIS、BOMBA、Kesihatan)是具有特定职责的紧急服务队伍。另外两个(Visitors 和 Malaysians)让你以普通人身份游玩。

| 队伍 | 职责 | 谁都可以加入? |
|------|--------------|------------------|
| Visitors | 普通平民,默认选项 | 可以 |
| Malaysians | 普通平民,为已验证的小组成员准备的装饰性队伍 | 仅限 Rank 3+ 的 MYSverse 小组成员 |
| POLIS | 警察:逮捕罪犯、巡逻、执行限速 | 可以 |
| BOMBA | 消防队:扑灭火灾、进行救援 | 可以 |
| Kesihatan | 医疗:治疗伤患、管理医院 | 可以 |

请参阅 [Teams](/sumaya/teams) 了解每个队伍的更多详情,包括初始工具。

**不知道选哪个?** 从 **Visitors** 开始。你可以立即尝试平民工作(钓鱼、mamak、棕油),之后熟悉环境后再切换到紧急服务队伍。

## 你的启动资金

每位新玩家都以以下资金开始:

- 钱包里 **100 Ringgit**
- 银行里 **1000 Ringgit**

这足够租几辆平民车辆或购买一间基本 tier-1 房屋。请参阅 [Economy](/sumaya/economy) 了解更多关于金钱的内容。

## 操作

Sumaya 使用标准的 Roblox 移动方式。在 PC 上:

| 动作 | 按键 |
|--------|-----|
| 移动 | W、A、S、D |
| Sprint | Left Shift(按住) |
| 跳跃 | Space |
| 与提示互动 | E(出现时) |
| 打开手机 | 屏幕上的手机按钮,或点击图标 |
| 聊天 | /(斜线)聚焦聊天栏 |

在手机和手柄上,屏幕上的按钮和标准 Roblox 手柄绑定按预期工作。

## 你的头五分钟

以下是你出生后立即该做的事:

### 1. 打开你的手机

手机是你的主要中枢。点击屏幕上的手机图标打开它。你会看到职业、quest、紧急呼叫、设置等的应用程式。请参阅 [Phone and UI](/sumaya/phone) 查看完整列表。

打开手机也算是你第一个 quest 的一个步骤。

### 2. 查看 quest 应用程式

你会自动开始 **Welcome to Sumaya** quest 链:12 个 quest 教你基础知识,并在你进行时支付 XP 和 Ringgit。每个 quest 都会准确告诉你接下来该做什么。

打开手机上的 **Quest** 应用程式查看你当前的目标。红色引导光束和 minimap 上的标记通常会把你指向正确的地点。请参阅 [Quests](/sumaya/quests) 查看完整链条。

### 3. 出现一辆车

走到任何**车辆 spawner 亭**(寻找散布在城里的提示)。挑选一辆车,租或买,然后开走。大多数入门平民车辆出现是免费的 - 较高档的车辆需要 Ringgit。

请参阅 [Vehicles](/sumaya/vehicles) 查看完整列表以及 spawner 的运作方式。

### 4. 开始一份平民工作

走到任何**职业站点**(钓鱼码头、mamak 摊位、棕榈种植园)并按互动提示打卡上班。你工作时会赚取 Ringgit 和 XP。工作在 [Careers overview](/sumaya/jobs) 中有描述。

如果你选择了紧急服务队伍(POLIS、BOMBA、Kesihatan),你不需要打卡 - 你一出生就在当班。

### 5. 探索

步行、驾驶或飞行。有地图、地标、商店、医院、警察局、消防局、海滩,以及许多邻里街区。Minimap(屏幕右上角)显示你周围有什么。

## HUD 概览

当你在世界中时,你会看到:

- **Minimap**(右上角):显示你的位置、附近的道路和建筑物、路径点,以及任务指向某处时的红色引导光束。点击可展开。
- **任务卡片**(左上角):显示你当前的目标(活动 quest 步骤、当前火灾、你正在治疗的病人等)。如果它挡路,可点击 chevron 折叠它。
- **钱包 / 银行余额**:在屏幕顶部附近可见。

## Sprint

按住 **Left Shift** 或 **Left Control** 进行 sprint。步行速度是 10 studs/s;sprint 将其加倍至 20。当前 build 中没有体力限制 - 你想 sprint 多久就 sprint 多久。(游戏代码中有完整的饥饿/体力/卫生系统,但会为每个角色初始化这些数值的 server 处理程序目前被禁用,所以它们在实际游戏中都不运行。)

进入车辆座位时,sprint 会自动关闭。

## 接下来呢?

- 选择一个 [team](/sumaya/teams) 和一个 [career](/sumaya/jobs)。
- 阅读 [phone and UI](/sumaya/phone),让你知道东西在哪。
- 查看 [quest chain](/sumaya/quests) 获取结构化的游玩指南。
- 当你准备好赚大钱时,查看 [economy](/sumaya/economy) 了解最赚钱的工作在哪。
