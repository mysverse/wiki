# 进度

Sumaya 拥有**双层 XP 系统**：你的每个动作都会获得**玩家 XP**（用于全局等级）和**领域 XP**（用于你正在从事的特定职业或服务）。本页解释一切如何运作 - 何时升级、职级如何嵌入，以及哪些策略最有效。

## 双层等级

### 玩家等级

- 全局账户等级，跨所有活动共享。
- 范围 **1 到 300**。
- 每个领域动作都会与你的领域 XP 并行增加你的玩家 XP。
- 较高的玩家等级解锁一些车辆（主要是 Polis）和任务链里程碑。
- 从 1 升级到 300 需要约 **450 万总 XP** - 这是一段漫长的磨炼，大多数玩家在远低于 300 时就会停下。

### 领域等级

- 每个职业的等级，对你工作的每个领域都是独立的。
- 每个领域现在范围 **1 到 100**（Bomba、Kesihatan、Polis、Fishing、Service 和 Agriculture/palm oil 都共享相同的长阶梯）。
- 你的领域等级决定该领域内的**职级**（头衔）。
- 较高的领域等级解锁团队特定车辆（例如 Bomba 领域 65 的 Sanica Tanker）。

### 它们如何一起升级

几乎每个游戏内动作都会给予**两种** XP，玩家 XP 和领域 XP，每种数量相同。例如，扑灭一个火焰块给予 5 玩家 XP 和 5 Bomba 领域 XP。

所以当你磨炼时，你的全局账户和特定领域都会并行上升。

### 任务

12-quest 探索链通过每个任务奖励支付 **3,750 XP**，**加上最后一个任务的一次性 2,000 XP 终章奖励**。完全完成 = **5,750 XP**。SR 支付类似：2,900 普通 + 1,500 终章 = 4,400 Ringgit。参见 [Quests](/sumaya/quests)。

## Kesihatan +15% XP 加成

伤害系统**只要服务器上有任何 Kesihatan 玩家就会自动激活**。激活时，每位玩家都会获得**玩家 XP 的 +15% 倍率**。加成适用于所有来源 - 犯罪、工作、火灾，任何给予玩家 XP 的事物。

领域 XP **不会**被加成 - 只有玩家 XP。所以加成帮助你磨炼全局账户，而不是你特定的职业。

这是让你的服务器上有医疗玩家活跃的强烈激励。开启时长时间磨炼会议效率高得多。

## 职业服装和装备

每个紧急服务团队（POLIS、BOMBA、Kesihatan）都有一个 **Careers** 应用流程，你可以在其中装备已解锁的服装和装备物品。这些物品由该团队的**领域等级**门控：

- **服装**会更换你角色的制服外观（例如 Nurse scrubs、Specialist firesuit、Ambulance Operator vest）。
- **装备物品**附加齿轮（腰带、头盔、面罩、雨衣）。

锁定的物品在 Careers 应用中显示 "Locked (Lv. X)"，直到你的领域等级达到阈值。门控示例：

- **POLIS**：L1 时 Duty Belt + Beret，L10 时 Reflective Vest，L20 时 Reflective Coat。
- **BOMBA**：L15 时 Respirator + SCBA，L20 时 EMRS，L30-40 时各种装备，L50 时 Senior Firesuit，L60 时 Command Helmet，L67 时 Calendar 服装。
- **Kesihatan**：L1 时 Nurse + Medical Mask，L15 时 Ambulance Operator + Medikal Cap，L20 时 N95，L25 时 Medikal Reflective Vest，L30 时 Scrubs + Face Shield。

## 升级策略

想要快速升级？以下是一些粗略的策略：

### 早期游戏（玩家等级 1-10）

- 完成任务链。它在等级 2、5 和 10 里程碑处前置加载，分块支付 300+500+800 = 1,600 XP。
- 选择一个平民领域来 clock in，同时完成其他任务步骤。
- 现在不要担心领域等级 - 专注于玩家 XP。

### 中期游戏（玩家等级 10-50）

- 如果还没有，请选择紧急服务。Kesihatan 的 +15% 加成对所有人都有帮助。
- Bomba 非常适合团队磨炼（更大的火灾，更多玩家 = 更快的里程碑奖励）。
- Polis 在非高峰时段很慢 - 你需要在线的罪犯。

### 后期游戏（玩家等级 50+）

- 棕榈油班次的班次结束 50 XP 奖励。
- 5 星逮捕，如果你能搞定的话（每次 400 XP）。
- 每 15 个块的 Bomba 火灾奖励 = 150 SR。

### 多领域策略

- 你在一个领域获得的所有东西都会并行升级你的玩家账户。
- 每个领域现在都封顶在等级 100 — 不再有"最短阶梯"领域。选择适合你喜欢做的事情的节奏。
- Fishing 是最被动的 — 在做其他事情时升级它。

## 多余 XP 结转

如果你有足够的 XP 在单个动作中多次升级（例如低等级时的大型 400 XP 逮捕），多余的部分会正确结转 - 你会一次性升级多个等级而不会损失 XP。

## 提示

- **跟踪 +15% 加成。** 长会议 + Kesihatan 激活 = 效率高得多。
- **不要忘记玩家 XP 与领域 XP 是分开的。** 如果你转换职业，两者都会继续增长。
- **职级是炫耀，不是游戏玩法。** 不要觉得有压力要去最大化你不喜欢的职级。
- **等级门控比看起来要少。** 大多数内容从等级 1 开始就可以使用。大门控是针对高级团队车辆的。
