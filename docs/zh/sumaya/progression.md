# 进度

Sumaya 拥有**双层 XP 系统**：你所做的每个动作都会获得**玩家 XP**（累积到你的全局等级）以及**行业 XP**（累积到你正在从事的特定职业或服务）。此页面将解释一切如何运作——何时升级什么、职级如何配合、以及哪些策略最有效。

## 两个层级

### 玩家等级

- 全局账户等级，与你所做的一切共享。
- 范围从 **1 到 300**。
- 每个行业动作都会与你的行业 XP 并行地增加你的玩家 XP。
- 更高的玩家等级会解锁一些车辆（大多是 Polis）以及任务链里程碑。
- 从 1 升到 300 大约需要 **450 万总 XP**——这是一段漫长的苦练，大多数玩家会在 300 之前很久就停下来。

### 行业等级

- 按职业划分的等级，每个你从事的行业相互独立。
- 范围依行业而定：
  - **Bomba、Kesihatan、Polis、Fishing、Service**：1 到 100。
  - **Agriculture（棕油）**：1 到 25。
- 你的行业等级驱动你在该行业内的**职级**（头衔）。
- 更高的行业等级会解锁团队专属车辆（例如 Bomba 65 级的 Sanica Tanker）。

### 它们如何一起升级

几乎每个游戏内动作都会**同时**给予玩家 XP 和行业 XP，两者数量相同。例如，扑灭一个火焰方块会给予 5 玩家 XP 和 5 Bomba 行业 XP。

所以当你苦练时，你的全局账户和特定行业会并行提升。

### 任务

12 任务发现链在每个任务奖励中总共支付 **3,750 XP**，**最后一个任务还有一次性的 2,000 XP 结局奖励**。完整完成 = **5,750 XP**。SR 支付类似：2,900 一般 + 1,500 结局 = 4,400 Ringgit。见[任务](/sumaya/quests)。

## Kesihatan +15% XP 加成

伤害系统**只要有任何 Kesihatan 玩家在 server 上就会自动启动**。当它启动时，每个玩家都会获得**玩家 XP 的 +15% 乘数**。此加成适用于所有来源——犯罪、工作、火灾，任何给予玩家 XP 的事物。

行业 XP **不会**被加成——只有玩家 XP。所以此加成有助于你的全局账户苦练，而不是你的特定职业。

这是一个强大的诱因，让你的 server 有医疗玩家活跃。当它开启时，长时间的苦练会议会高效许多。

## 职业装束与配件

每个紧急服务团队（POLIS、BOMBA、Kesihatan）都有 **Careers** app 流程，你可以在那里装备已解锁的装束和配件。这些物品由你该团队的**行业等级**管控：

- **装束** (Outfits) 切换你角色的制服外观（例如 Nurse scrubs、Specialist firesuit、Ambulance Operator vest）。
- **配件** (Kit items) 附加装备（腰带、头盔、口罩、雨衣）。

锁定的物品在 Careers app 中会显示 "Locked (Lv. X)"，直到你的行业等级达到门槛。示例门槛：

- **POLIS**：L1 的 Duty Belt + Beret、L10 的 Reflective Vest、L20 的 Reflective Coat。
- **BOMBA**：L15 的 Respirator + SCBA、L20 的 EMRS、L30-40 的各种配件、L50 的 Senior Firesuit、L60 的 Command Helmet、L67 的 Calendar outfit。
- **Kesihatan**：L1 的 Nurse + Medical Mask、L15 的 Ambulance Operator + Medikal Cap、L20 的 N95、L25 的 Medikal Reflective Vest、L30 的 Scrubs + Face Shield。

## 升级策略

想快速升级？以下是粗略策略：

### 游戏初期（玩家等级 1-10）

- 做任务链。它前段很重，包含 2、5 和 10 级里程碑，分段支付 300+500+800 = 1,600 XP。
- 选择一个平民行业来 clock in，同时完成其他任务步骤。
- 暂时不用担心行业等级——专注于玩家 XP。

### 游戏中期（玩家等级 10-50）

- 如果还没选，请选一个紧急服务。Kesihatan 的 +15% 加成对所有人都有帮助。
- Bomba 非常适合团体苦练（更大的火灾、更多玩家 = 更快的里程碑奖励）。
- 非高峰时段的 Polis 很慢——你需要罪犯在线。

### 游戏后期（玩家等级 50+）

- 棕油班次的下班 50 XP 奖励。
- 若能做到的 5 星逮捕（每个 400 XP）。
- 每 15 个方块的 Bomba 火灾奖励 = 150 SR。

### 多行业策略

- 你在某个行业赚取的一切都会并行提升你的玩家账户。
- 将 Agro 练满（25 级）是最短的阶梯——适合达成"首次练满"成就。
- Fishing 是最被动的——做其他事情时顺便练它。

## 多余 XP 结转

如果你在一个动作中拥有足够的 XP 多次升级（例如低等级时的 400 XP 大逮捕），多余的 XP 会正确结转——你会一次性跨越多个等级而不会丢失 XP。

## 小贴士

- **追踪 +15% 加成。** 长时间会议 + Kesihatan 活跃 = 高效许多。
- **别忘记玩家 XP 与行业 XP 是分开的。** 如果你转换职业，两者都会继续成长。
- **职级是炫耀用的，不是 gameplay。** 不要被迫练满你不喜欢的职级。
- **等级门槛比看起来的少。** 大多数内容从 1 级就可用。主要门槛是给高阶团队车辆的。
