# 队伍

Sumaya 有五个队伍，每个队伍都有独特的角色、工具和职责。你可以随时通过主菜单的**队伍**标签切换队伍。

## 普通队伍

### 访客（Visitors）

<!-- TODO: 添加访客队伍截图 -->
<!-- ![访客](/images/sumaya/teams/visitors.png) -->

所有新玩家的默认队伍。

- **颜色：** 绿色
- **要求：** 无（自动分配）
- **工具：** 仅标准初始工具
- **描述：** 访客可以自由探索城市、使用基本工具、钓鱼、从事平民工作并与世界互动。这是你首次加入 Sumaya 时所在的队伍。

### 马来西亚公民（Malaysians）

- **颜色：** 橙色
- **要求：** MYSverse Roblox 群组成员（需要最低等级）
- **工具：** 仅标准初始工具
- **描述：** 为经过验证的 MYSverse 社区成员设立的队伍。马来西亚公民可以访问额外的出生点，并被认可为社区成员。

## 紧急服务

### POLIS（警察）

加入 POLIS 担任执法人员，维护 Sumaya 的秩序。

- **颜色：** 蓝色
- **要求：** 无
- **工作部门：** Token_Polis
- **职业发展：** 是 — 晋升以解锁新装备和职责

**提供的工具：**

| 工具 | 描述 |
|------|------|
| **Taser X26P** | 用于制服嫌疑人的非致命武器 |
| **测速枪** | 监测车辆速度并设置测速陷阱 |
| **哨子** | 向市民和其他警员发出警报 |
| **手铐** | 逮捕和拘留违法者 |

**职责：**

- 在城市巡逻并响应事件
- 使用测速枪和测速陷阱系统监控交通速度
- 使用手铐逮捕违法者
- 向违反规则的玩家发出违规通知
- 与其他紧急服务（BOMBA 和 Kesihatan）协调

POLIS 成员还可以使用**测速陷阱 GUI**，显示捕获的车辆速度，帮助警员识别超速驾驶者。

加入 BOMBA 响应火灾事件并保护城市。

- **要求：** 无
- **工作部门：** Token_Bomba
- **职业发展：** 是

**提供的工具：**

| 工具 | 描述 |
|------|------|
| **灭火器** | 用于扑灭小型火灾的便携式灭火器 |

**消防水管系统：**

BOMBA 成员还可以使用基于消防栓的消防水管系统：

1. 在事件附近找到**消防栓**。
2. 从消防栓连接**供水管**。
3. 使用**水管出口**将水对准火源。

**职责：**

- 响应城市各处出现的火灾事件
- 使用灭火器和消防水管系统灭火
- 通过专用 BOMBA GUI 跟踪活跃事件
- 与 Kesihatan 协调处理受伤人员

::: tip 可放置工具
BOMBA 和 POLIS 队伍可以使用可放置工具，如锥桶、标志和路障，可放置在事件现场周围。
:::

### Kesihatan（医疗服务）

加入 Kesihatan 提供医疗护理并响应健康紧急事件。

- **颜色：** 白色
- **要求：** 无
- **工作部门：** Token_Kesihatan
- **职业发展：** 是

**提供的工具：**

| 工具 | 描述 |
|------|------|
| **急救箱** | 在现场诊断患者的伤情 |
| **绷带** | 治疗开放性伤口和撕裂伤 |
| **夹板** | 稳定骨折 |

**医院设备：**

Kesihatan 成员可以操作医院的设备：

- **MRI 机器：** 进行全身扫描以诊断内部损伤。使用专用 MRI GUI 界面。
- **X 光机器：** 对特定身体部位进行定向扫描。使用专用 X 光 GUI 界面。
- **病床：** 收治患者进行治疗和休息。医院有 12 张病床。

**职责：**

- 响应城市各处受伤的玩家
- 使用急救箱诊断伤情
- 用绷带治疗伤口，用夹板治疗骨折
- 操作医院的 MRI 和 X 光机器
- 监控伤害和碰撞通知系统

**伤害系统：**

Sumaya 有车辆碰撞系统，可能导致玩家受伤。当碰撞发生时：

1. **伤害通知**会发送给附近的 Kesihatan 成员。
2. Kesihatan 响应并使用急救箱**诊断**伤情。
3. 施用适当的治疗（绷带治疗伤口，夹板治疗骨折）。
4. 对于严重伤害，患者可能需要被送往**医院**进行 MRI 或 X 光扫描。

## 队伍机制

### 切换队伍

切换队伍的方法：

1. 打开**主菜单**。
2. 导航到**队伍**标签。
3. 选择你想加入的队伍。
4. 如果你满足要求，你将以新队伍的颜色和工具重新出生。

::: warning 群组要求
**马来西亚公民**队伍要求你是 MYSverse Roblox 群组的成员，并达到最低等级。如果你不满足要求，将无法加入该队伍。
:::

### 职业发展

三个服务队伍（POLIS、BOMBA 和 Kesihatan）各有自己的**职业发展系统**：

- 每个职业都有专门的**工作部门**和等级。
- 通过执行队伍职责和完成任务来晋升。
- 通过主菜单的**职业**标签访问职业信息和自定义选项。
- 随着职业发展装备不同的**制服**和**装备物品**。

### 队伍出生点

每个队伍都有指定的**出生位置**和**车辆出生点**：

- **POLIS：** 专用警车出生点，配备执法车辆。
- **BOMBA：** 消防站附近的专用消防车出生点。
- **Kesihatan：** 医院附近的专用医疗车辆出生点。
- **访客/马来西亚公民：** 通用出生位置和民用车辆出生点。

### 工资

服务角色的队伍成员会定期收到**工资**。工资 GUI 显示：

- 基本工资金额
- 奖金收入
- 任何扣除
- 总支出

工资直接存入你的账户，当付款准备好时你会收到通知。
