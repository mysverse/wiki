# 紧急服务概览

Sumaya 有三支紧急服务队伍：

- [Bomba](/sumaya/bomba) - 消防队
- [Kesihatan](/sumaya/kesihatan) - 医疗 / 护理人员
- [Polis](/sumaya/polis) - 警察

选择其中一支队伍后，您一 spawn 就立即处于 **值勤状态**。与平民职业（需要通过手机打卡）不同，紧急服务是您的身份，而非您正在做的事。

## 紧急服务与平民工作有何不同？

| | 平民职业 | 紧急服务 |
|---|------------------|---------------------|
| 加入方式 | 手机上的 Careers 应用 | 主菜单队伍选择 |
| 最高领域等级 | 25（农业）或 100（捕鱼 / 服务业） | 100 |
| 每个动作 XP | 随等级调整 | 每个动作固定 |
| 工具 | 值班开始时发放 | spawn 时发放 |
| 需要打卡？ | 是 | 否，始终值勤 |
| 等级限制的载具？ | 仅拖网渔船 | 是，有几辆限制在领域等级 15、20、35、40、45、55、65 |

紧急服务也拥有最详细的阶级制度 - Bomba 和 Kesihatan 有 13 个阶级，Polis 有 16 个，带有正式的马来西亚头衔。

## 我应该先试哪一个？

快速指南：

- **您喜欢动作与追击：** [Polis](/sumaya/polis)。追捕嫌犯、设置测速陷阱、进行逮捕。server 里有罪犯时最动感。
- **您喜欢团队合作与大场面：** [Bomba](/sumaya/bomba)。火灾是团体事件。多名消防员共同处理一起事件是升级最快的方式。
- **您喜欢帮助个人：** [Kesihatan](/sumaya/kesihatan)。每位病人都是一对一：诊断、治疗、继续前进。在您的受伤系统启动时也会给整个 server 带来 **+15% player-XP** 加成。

## 共同点

三种紧急服务都有一些共同机制：

### 阶级与等级

您的 **领域等级** 追踪您在特定服务中的进度。您在里程碑等级时升阶（例如 Bomba 的 6、11、16、21... 级）。阶级 **仅为头衔** - 不会锁定玩法。只有您的 **等级** 会限制如载具等事物。

三者的最高领域等级都是 **100**。阶级会在此之前停止晋升（例如 Polis 阶级在 89 级封顶），所以最后一段纯粹是炫耀。

### Ringgit 奖励进入您的钱包

与平民职业（款项汇入您的银行账户）不同，紧急服务奖励直接付入您的 **钱包**，但基于犯罪的付款除外。这意味着您可以立即消费而无需前往 ATM。

### 每 N 次行动的奖金

每项服务都有 **里程碑奖金**：

- Bomba：每 **15** 个火焰方块奖励 **150 Ringgit**。
- Kesihatan：每 **10** 次治疗奖励 **150 Ringgit**。
- Polis：每 **5** 次逮捕奖励 **200 Ringgit**。

这些奖金会在您每次行动奖励之上叠加。

### 通缉状态降级

游戏被设定为当紧急服务玩家（BOMBA、Kesihatan、POLIS）进入 1 星通缉时自动降级，但根据目前的验证该事件的监听器并未连接 - 即使犯罪后您也会 **留在您的紧急队伍**。降级 bindable 触发时没有处理程序。不要依赖游戏将您换回平民身份；在做任何违法行为之前手动切换队伍。

## Kesihatan 的全局 XP 加成

这一点很独特：每当 **任何 Kesihatan 玩家在 server 上**，受伤系统就会 **自动启动**，并且 **server 上的每位玩家在启动期间都获得 +15% 玩家等级 XP**（领域 XP 不受影响）。最后一名 Kesihatan 玩家离开时立即停用。如果您在刷分，这是个大事 - 在长时间游戏前留意加成指示器。

## 紧急载具

每项服务都有自己的载具，随着您的领域等级上升而解锁。完整目录请见 [载具](/sumaya/vehicles)，或查看各服务页面的队伍专属清单。

紧急载具有 **警笛与警灯**（ELS）。驾驶紧急载具时开启警笛可让您暂时免除交通违规 - 当您的警笛启动时，您不会因超速或碰撞被开罚单。

## 999 紧急电话

平民可以从手机拨打 **999** 请求紧急服务。来电会显示在 **MDT（Mobile Data Terminal）** 应用中，所有值勤的紧急服务玩家都能看到。您可以：

- 在您的迷你地图上查看来电者的位置。
- 阅读他们附加的任何自定义备注。
- 设置路径点以导航前往。
- 将来电标记为已回应 / 已关闭。

这是平民召唤您的主要方式。MDT 在手机的应用列表中 - 它仅向紧急服务队伍的玩家显示。参见 [电话与 UI](/sumaya/phone)。
