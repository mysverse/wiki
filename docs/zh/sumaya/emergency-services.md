# 紧急服务概览

Sumaya 有三支紧急服务队伍:

- [Bomba](/sumaya/bomba) - 消防队
- [Kesihatan](/sumaya/kesihatan) - 医疗 / 护理
- [Polis](/sumaya/polis) - 警察

选择其中一支队伍会让你在 spawn 后立即处于**当班**状态。与平民职业(需要通过手机打卡上班)不同,紧急服务就是你的身份,而不是你正在做的事情。

## 紧急服务与平民工作有什么不同?

| | 平民职业 | 紧急服务 |
|---|------------------|---------------------|
| 如何加入 | 手机上的 Careers 应用 | 主菜单队伍选择 |
| 最高领域等级 | 100 (每个平民领域 — fishing、service、agro) | 100 |
| 每次行动的 XP | 固定 (Service 每位顾客 6,Agro 每个水果 6,Trawl 每条鱼 2,Rod 每条鱼 15) — 无等级缩放 | 每次行动固定 (Bomba 每块 5,Kesihatan 每次治疗 15-40,Polis 50-400 按星级缩放) |
| 工具 | 上班时给予 | spawn 时给予 |
| 需要打卡? | 需要 | 不需要,始终当班 |
| 等级限制的载具? | 仅 trawler | 是,数辆在领域等级 15、20、35、40、45、55、65 解锁 |

紧急服务也拥有最详细的军衔系统 - Bomba 和 Kesihatan 有 13 个军衔,Polis 有 16 个,均使用正式的马来西亚头衔。

## 我应该先尝试哪一个?

快速指南:

- **你喜欢动作和追捕:** [Polis](/sumaya/polis)。追逐嫌犯、设置测速陷阱、登记逮捕。在 server 有犯罪分子时最为动态。
- **你喜欢团队合作和重大时刻:** [Bomba](/sumaya/bomba)。火灾是团体事件。多名消防员处理同一事件是升级最快的方式。
- **你喜欢帮助个人:** [Kesihatan](/sumaya/kesihatan)。每位病人都是一对一:诊断、治疗、继续。在你的伤势系统启用时,还会为整个 server 提供 **+15% 玩家 XP** 加成。

## 共同点

所有三种紧急服务共享一些机制:

### 军衔与等级

你的**领域等级**追踪你在特定服务中的进度。你在里程碑等级升衔 (例如 Bomba 的等级 6、11、16、21...)。军衔只是**头衔** - 它们不会锁定玩法。只有你的**等级**才会限制载具等内容。

最高领域等级对三者皆为 **100**。军衔在那之前就停止晋升 (例如 Polis 军衔在等级 89 封顶),所以最后的冲刺只是炫耀。

### Ringgit 奖励直接进入你的钱包

与平民职业 (支付到你的银行账户) 不同,紧急服务奖励直接支付到你的**钱包**,但基于犯罪的支付除外。这意味着你可以立即花费,无需前往 ATM。

### 每 N 次行动的奖金

每种服务都有**里程碑奖金**:

- Bomba:每 **15** 个火灾区块奖励 **150 Ringgit**。
- Kesihatan:每 **10** 次治疗奖励 **150 Ringgit**。
- Polis:每 **5** 次逮捕奖励 **200 Ringgit**。

这些奖金叠加在你的每次行动奖励之上。

### 通缉状态下的降级

游戏被设置为当紧急服务玩家 (BOMBA、Kesihatan、POLIS) 进入 1 星通缉时自动降级,但根据当前验证,该事件的监听器并未连接 - 即使在犯罪后,你仍会**留在你的紧急队伍中**。降级 bindable 触发但无处理程序。不要依赖游戏将你切换回平民;在做任何违法事情之前手动切换队伍。

## Kesihatan 的全局 XP 加成

这是独特的:**只要 server 上有任何 Kesihatan 玩家**,伤势系统就会**自动启用**,并且**server 上的每位玩家都获得 +15% 玩家等级 XP** (领域 XP 不受影响)。最后一名 Kesihatan 玩家离开时立即停用。如果你正在 grind,这是一件大事 - 在长时间 session 之前注意加成指示器。

## 紧急载具

每种服务都有自己的载具,随着你的领域等级提升而解锁。完整目录请参阅 [Vehicles](/sumaya/vehicles),或查看各服务页面以了解队伍专属列表。

紧急载具配备**警笛和灯光** (ELS)。在驾驶紧急载具时打开警笛会让你暂时免除交通违规 - 在警笛激活期间,你不会因超速或撞车被开罚单。

## 999 紧急呼叫

平民可以从手机拨打 **999** 请求紧急服务。呼叫会显示在 **MDT (Mobile Data Terminal)** 应用上,所有当班的紧急服务玩家都能看见。你可以:

- 在你的 minimap 上查看呼叫者的位置。
- 阅读他们附上的任何自定义注释。
- 设置导航点前往那里。
- 将呼叫标记为已响应 / 已关闭。

这是平民召唤你的主要方式。MDT 在手机的应用列表中 - 仅对紧急服务队伍的玩家显示。请参阅 [Phone and UI](/sumaya/phone)。
