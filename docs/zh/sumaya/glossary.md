# 术语表

马来西亚（马来）词汇和 Sumaya 专属游戏术语的参考列表。

## 马来西亚词汇

这些来自日常马来西亚／马来词汇。你在整个游戏中都会看到它们。

| 术语 | 含义 | 游戏用途 |
|------|---------|------------|
| **Ringgit** | 马来西亚货币（现实中：MYR）。 | Sumaya 的游戏内货币。在 UI 中缩写为 **SR**（Sumaya Ringgit）。 |
| **Bomba** | 消防队（源自葡萄牙语的马来词）。 | 紧急服务队伍之一 - 消防员。 |
| **Kesihatan** | 健康／医疗。 | 医疗／护理人员紧急服务队伍。 |
| **Polis** | 警察。 | 执法紧急服务队伍。缩写为 POLIS。 |
| **Jom** | "走吧"（非正式）。 | 从主菜单进入游戏世界的按钮。 |
| **Daerah** | 区／地区。 | Sumaya 地图上的地区。 |
| **Mamak** | 印度穆斯林餐厅，马来西亚的一种机构。 | 两个服务业场所之一（另一个是快餐）。 |
| **Kedai** | 店铺／店。 | 一般店铺招牌。 |
| **Kampung** | 村庄。 | Sumaya 中的住宅区；也是传统高脚木屋模板的名称（"婆罗洲"房屋属于 kampung 风格）。 |
| **Pasaram** | 市场／集市。 | 市场地点。 |
| **Abang Samseng** | "流氓大哥" - 街头恶棍，小罪犯。 | 开始非法工作（如动物走私）时你与之交谈的犯罪任务 NPC。 |
| **Sadaqa** | 慈善捐赠／施舍（源自阿拉伯语，在马来语中广泛使用）。 | Wallet 工具上玩家对玩家捐赠提示的标签。 |
| **Nelayan** | 渔夫。 | "Nelayan Sdn. Bhd." 是捕鱼职业的薪水支付方（意为 "Fisherman Co. Ltd."）。 |
| **Sdn. Bhd.** | "Sendirian Berhad" 的缩写 - 马来西亚相当于 "Ltd." 或 "Pte Ltd"。 | 出现在薪水通知中。 |
| **Jabatan** | 部门／当局。 | 用于现实中的等级头衔，如 "Bomba MYSverse"（消防与救援部）。 |
| **Kementerian** | 部。 | "KKN" 代表 **Kementerian Kesihatan Nasional** - Kesihatan 车辆上的救护车标志。 |
| **Jalan** | 路／街。 | 出现在街道名称中，如 "Jalan Triden"、"Jalan Bawang"。 |
| **Lorong** | 巷／小路。 | 较小的街道，如 "Lorong Pais"、"Lorong Kicai"。 |

## 游戏专属术语

| 术语 | 定义 |
|------|------------|
| **SR** | Sumaya Ringgit - 游戏内货币的缩写。与 "Ringgit" 相同。 |
| **Wallet** | 你可立即使用的现金。可被偷窃。 |
| **Bank** | 你的安全储蓄。无法被偷窃。通过 ATM 访问。 |
| **Offence points** | 因犯罪累积的分数。总分决定你的通缉星级。 |
| **Wanted stars** | 你的犯罪状态，1 到 5。越高 = 越严重。 |
| **Wanted decay** | 5 分钟内不犯下新罪行就会失去所有 offence points。 |
| **Passive mode** | 一种可切换状态，你无法受伤也无法犯罪。 |
| **Passive lock** | 对 passive mode 切换的临时禁用，在某些罪行后施加。 |
| **LTAA** | "Leave To Avoid Arrest" - 在犯罪期间登出而获得的额外监狱时间。 |
| **Team** | 你的角色（Visitors、Malaysians、POLIS、BOMBA、Kesihatan）。 |
| **Sector** | 工作类型：Fishing、Service、Agro、Bomba、Polis、Kesihatan。 |
| **Rank** | 你在某个 sector 内的声望头衔（如 "Fire Officer"）。仅为装饰。 |
| **Token** | sector 的内部名称（如 "Token_Bomba"）。你在游戏中很少看到这个。 |
| **Plot** | 邻里中你可以盖房子的一块地。每位玩家一块。 |
| **Tier**（房屋） | 房屋质量等级：Tier 1（便宜）、Tier 2（中等）、Tier 3（高级）。 |
| **Tier**（trawler） | 渔船质量：Tier 1（免费）、Tier 2（25k）、Tier 3（37k）。 |
| **ELS** | Emergency Lighting System - 紧急车辆上的灯光和警报器。 |
| **LFRT** | Light Fire Rescue Tender - 较小的消防车。 |
| **FRT** | Fire Rescue Tender - 较重的消防车。 |
| **MPV** | Multi-Purpose Vehicle - 某些警车的车身款式。 |
| **MDT** | Mobile Data Terminal - 你手机上的警察／紧急调度应用。 |
| **999** | 马来西亚紧急服务电话号码（类似美国 911 或英国 999）。平民在 Emergency999 应用中拨打。 |
| **Minimap** | 右上角显示你所在区域的 HUD 地图。 |
| **Task card** | 左上角显示你当前目标的 HUD 面板。 |
| **Waypoint** | minimap 上的标记位置。可由游戏设置（任务、MDT 来电）或自定义（长按放置）。 |
| **Guidance beam** | 从你的角色指向你激活 waypoint 的彩色 3D 光束。 |
| **Stud** | Roblox 的距离单位。大约 1 英尺／30 厘米。 |
| **KKN** | Kementerian Kesihatan Nasional - 国家卫生部。在救护车上可见。 |
| **Slim Jim** | 用于偷车的工具。一次性使用。 |
| **Pickpocket Knife** | 用于偷窃玩家的工具。 |
| **Stolen plate** | 车辆被偷时其车牌会被标记。POLIS 可以通过 MDT 查询车牌。 |
| **Paycheck** | 职业轮班结束时发放的 Ringgit 报酬，随通知一起送达。 |
| **Injury system** | 允许玩家遭受非致命伤害并需要 Kesihatan 治疗的医疗机制。激活时，让所有人获得 +15% 玩家 XP。 |
| **TBI** | Traumatic Brain Injury - 最严重的伤害类型，需要卧床休息。 |
| **Drop-off zone** | 偷车贼交付偷来的车辆出售的地点，或动物走私者交付非法货物的地点。 |
| **Nearby officer bonus** | 逮捕奖励的 50% 给予任何在逮捕官员 50 studs 范围内的 POLIS 玩家。 |
