# 术语表

马来西亚（马来语）词汇与 Sumaya 专属游戏术语的参考列表。

## 马来西亚词汇

这些词汇来自日常的马来西亚/马来语词汇。你将在整个游戏中看到它们。

| 术语 | 含义 | 游戏中的用法 |
|------|---------|------------|
| **Ringgit** | 马来西亚货币（现实世界：MYR）。 | Sumaya 的游戏内货币。在 UI 中缩写为 **SR**（Sumaya Ringgit）。 |
| **Bomba** | 消防队（源自葡萄牙语的马来词）。 | 紧急服务团队之一 - 消防员。 |
| **Kesihatan** | 健康 / 医疗。 | 医疗 / 护理人员紧急服务团队。 |
| **Polis** | 警察。 | 执法紧急服务团队。缩写为 POLIS。 |
| **Jom** | "走吧"（非正式）。 | 从主菜单进入世界的按钮。 |
| **Jelajah** | "探索" / "旅程"。 | 手机上的发现日志应用 — 追踪你已找到的地标（POI）以及仍被锁定的地标。 |
| **Ditemui** | "找到" / "发现"。 | 用于 Jelajah 应用的进度标签（`<n> / <total> ditemui`）。 |
| **Baca Maklumat** | "阅读信息"。 | 已发现的 Jelajah POI 详情页上的 CTA 按钮 — 重新打开地标的信息图。 |
| **Belum** | "尚未"。 | Jelajah 中针对你尚未发现的地标的筛选标签。 |
| **Semua** | "全部"。 | Jelajah 中针对完整 POI 列表的筛选标签。 |
| **Daerah** | 区域 / 地区。 | Sumaya 地图的区域。 |
| **Mamak** | 印度穆斯林餐馆，一种马来西亚的传统机构。 | 服务行业的两个场所之一（另一个是快餐店）。 |
| **Kedai** | 商店 / 店铺。 | 通用商店招牌。 |
| **Kampung** | 村庄。 | Sumaya 中的住宅区；也是传统高脚木屋模板的名称（"Bornean" 房屋是 kampung 风格）。 |
| **Pasaram** | 市场 / 集市。 | 市场位置。 |
| **Abang Samseng** | "黑帮兄弟" - 街头硬汉、小罪犯。 | 你在开始非法工作（例如动物走私）时与之交谈的犯罪任务 NPC。 |
| **Sadaqa** | 慈善捐赠 / 施舍（源自阿拉伯语，在马来语中广泛使用）。 | Wallet 工具上玩家对玩家捐赠提示的标签。 |
| **Nelayan** | 渔夫。 | "Nelayan Sdn. Bhd." 是渔业职业的薪资发放方（意为 "渔夫有限公司"）。 |
| **Sdn. Bhd.** | "Sendirian Berhad" 的缩写 - 马来西亚相当于 "Ltd." 或 "Pte Ltd"。 | 出现在薪资通知上。 |
| **Jabatan** | 部门 / 当局。 | 用于现实世界的等级头衔，例如 "Bomba MYSverse"（消防与救援部门）。 |
| **Kementerian** | 部委。 | "KKN" 代表 **Kementerian Kesihatan Nasional** - Kesihatan 车辆上的救护车标志。 |
| **Jalan** | 道路 / 街道。 | 出现在街道名称中，如 "Jalan Triden"、"Jalan Bawang"。 |
| **Lorong** | 巷 / 小路。 | 较小的街道，如 "Lorong Pais"、"Lorong Kicai"。 |

## 游戏专属术语

| 术语 | 定义 |
|------|------------|
| **SR** | Sumaya Ringgit - 游戏内货币的缩写。与 "Ringgit" 相同。 |
| **Wallet** | 你可以立即使用的现金。可被扒手偷走。 |
| **Bank** | 你的安全储蓄。无法被偷走。通过 ATM 访问。 |
| **Offence points** | 因犯罪累积的分数。总数决定你的通缉星等。 |
| **Wanted stars** | 你的犯罪状态，1 到 5。越高 = 越严重。 |
| **Wanted decay** | 通过 5 分钟内不犯新罪而失去所有 offence points。 |
| **Passive mode** | 一种可切换的状态，在该状态下你不会受到伤害也不能犯罪。 |
| **Passive lock** | 在某些罪行后施加的对 passive-mode 切换的临时禁用。 |
| **LTAA** | "Leave To Avoid Arrest" - 你在犯罪过程中登出而获得的额外监禁时间。 |
| **Team** | 你的角色（Visitors、Malaysians、POLIS、BOMBA、Kesihatan）。 |
| **Sector** | 一种工作类型：Fishing、Service、Agro、Bomba、Polis、Kesihatan。 |
| **Rank** | 你在某个 sector 内的声望头衔（例如 "Fire Officer"）。仅装饰用。 |
| **Token** | sector 的内部名称（例如 "Token_Bomba"）。在游戏中很少看到。 |
| **Plot** | 在邻里中你可以建造房子的一块土地。每个玩家一块。 |
| **Tier**（房屋） | 房屋质量等级：Tier 1（便宜）、Tier 2（中等）、Tier 3（高端）。 |
| **Tier**（拖网渔船） | 渔船质量：Tier 1（免费）、Tier 2（25k）、Tier 3（37k）。 |
| **ELS** | Emergency Lighting System - 应急车辆上的灯光和警报器。 |
| **LFRT** | Light Fire Rescue Tender - 较小型消防车。 |
| **FRT** | Fire Rescue Tender - 较重型消防车。 |
| **MPV** | Multi-Purpose Vehicle - 某些警车的车身样式。 |
| **MDT** | Mobile Data Terminal - 你手机上的警察/紧急调度应用。 |
| **999** | 马来西亚紧急服务电话号码（类似于美国的 911 或英国的 999）。平民在 Emergency999 应用中拨打。 |
| **Minimap** | 显示你所在区域的右上 HUD 地图。 |
| **Task card** | 显示你当前目标的左上 HUD 面板。 |
| **Waypoint** | minimap 上标记的位置。可以由游戏设置（任务、MDT 呼叫）或自定义（长按放置）。 |
| **Guidance beam** | 从你的角色指向活跃 waypoint 的彩色 3D 光束。 |
| **Stud** | Roblox 的距离单位。大约 1 英尺 / 30 厘米。 |
| **KKN** | Kementerian Kesihatan Nasional - 国家卫生部。在救护车上看到。 |
| **Slim Jim** | 用于劫车的工具。一次性使用。 |
| **Pickpocket Knife** | 用于扒窃玩家的工具。 |
| **Stolen plate** | 当车辆被劫时，其车牌会被标记。POLIS 可以通过 MDT 查询车牌。 |
| **Paycheck** | 来自职业的轮班结束时的 Ringgit 支付，伴随通知发放。 |
| **Injury system** | 让玩家遭受需要 Kesihatan 治疗的非致命伤害的医疗机制。激活时，给所有人 +15% player-XP。 |
| **TBI** | Traumatic Brain Injury - 最严重的伤害类型，需要卧床休息。 |
| **Drop-off zone** | 劫车者交付被盗车辆出售的位置，或动物走私者交付其非法商品的位置。 |
| **Nearby officer bonus** | 逮捕奖励的 50% 给予在逮捕官员 50 stud 范围内的任何 POLIS 玩家。 |
| **Combat timer / "ENGAGED"** | 当 taser 击中或拘留成功时，对嫌疑人开启的 30 秒交战窗口。在它运行时，Reset 和主菜单 / team-switch 被锁定，死亡或离开游戏将路由到时长加倍的 **combat-evasion jail**。无论哪种情况，逮捕官员都保留功劳。参见[犯罪与法律](/sumaya/crime-and-law)。 |
| **Reset cooldown / "Respawning..."** | 当你在战斗外按下 Reset 时运行的 12 秒可取消倒计时。干净地重新加载角色（无死亡，无掉落工具）。如果你被电击、逮捕或离开游戏，将自动取消。 |
| **Citizen level** | 你的全局玩家等级，显示在右上角 Status 小工具上为 `Citizen <N>`。与 sector 等级不同（后者是按职业的）。 |
| **Status widget** | 显示 Citizen 等级 + 活跃 sector 的两行右上 HUD 面板。通过点击顶栏上的 Status 图标隐藏。 |
| **TopBar icons** | 屏幕中上方的小型 Map / Tasks / Status / Open Menu 图标。点击切换每个 HUD；点击 **Open Menu**（房子图标）以在游戏中重新打开主菜单而不死亡。 |
