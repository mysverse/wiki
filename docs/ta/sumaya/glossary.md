# சொற்களஞ்சியம்

மலேசிய (மலாய்) சொற்கள் மற்றும் Sumaya-க்கான குறிப்பிட்ட விளையாட்டு சொற்களின் குறிப்புப் பட்டியல்.

## மலேசிய சொற்கள்

இவை அன்றாட மலேசிய/மலாய் சொற்களஞ்சியத்திலிருந்து வருகின்றன. விளையாட்டின் முழுவதிலும் இவற்றை நீங்கள் காண்பீர்கள்.

| சொல் | பொருள் | விளையாட்டில் பயன்பாடு |
|------|---------|------------|
| **Ringgit** | மலேசிய நாணயம் (உண்மை உலகம்: MYR). | Sumaya-வின் விளையாட்டு உள் நாணயம். UI-இல் **SR** (Sumaya Ringgit) என சுருக்கப்பட்டுள்ளது. |
| **Bomba** | தீயணைப்புப் படை (போர்த்துகீசிய தோற்றம் கொண்ட மலாய் சொல்). | அவசர சேவைக் குழுக்களில் ஒன்று - தீயணைப்பு வீரர்கள். |
| **Kesihatan** | சுகாதாரம் / மருத்துவம். | மருத்துவ / துணை மருத்துவ அவசர சேவைக் குழு. |
| **Polis** | காவல்துறை. | சட்ட அமலாக்க அவசர சேவைக் குழு. POLIS என சுருக்கப்பட்டுள்ளது. |
| **Jom** | "செல்வோம்" (சாதாரண). | முதன்மை மெனுவிலிருந்து உலகில் நுழைவதற்கான பொத்தான். |
| **Jelajah** | "ஆராய" / "பயணம்". | தொலைபேசியில் கண்டுபிடிப்பு பத்திரிகை செயலி — நீங்கள் கண்டுபிடித்த அடையாளங்கள் (POI-கள்) மற்றும் இன்னும் பூட்டப்பட்டவற்றைக் கண்காணிக்கிறது. |
| **Ditemui** | "கண்டுபிடிக்கப்பட்டது" / "கண்டறியப்பட்டது". | Jelajah செயலியின் முன்னேற்ற முத்திரையில் பயன்படுத்தப்படுகிறது (`<n> / <total> ditemui`). |
| **Baca Maklumat** | "தகவலைப் படிக்கவும்". | கண்டுபிடிக்கப்பட்ட Jelajah POI-இன் விவர திரையில் CTA பொத்தான் — அடையாளத்தின் தகவல் வரைகலையை மீண்டும் திறக்கிறது. |
| **Belum** | "இன்னும் இல்லை". | நீங்கள் இன்னும் கண்டுபிடிக்காத அடையாளங்களுக்கான Jelajah-இல் வடிகட்டி முத்திரை. |
| **Semua** | "அனைத்தும்". | முழு POI பட்டியலுக்கான Jelajah-இல் வடிகட்டி முத்திரை. |
| **Daerah** | மாவட்டம் / பகுதி. | Sumaya வரைபடத்தின் பகுதிகள். |
| **Mamak** | இந்திய-முஸ்லிம் உணவகம், ஒரு மலேசிய நிறுவனம். | இரண்டு சேவைத்துறை இடங்களில் ஒன்று (மற்றொன்று துரித உணவு). |
| **Kedai** | கடை / வணிகம். | பொது கடை அடையாளம். |
| **Kampung** | கிராமம். | Sumaya-இல் குடியிருப்புப் பகுதிகள்; பாரம்பரியமான தூண்களில் கட்டப்பட்ட மரவீட்டுத் தோற்றத்திற்கான பெயரும் ("Bornean" வீடுகள் kampung-பாணி). |
| **Pasaram** | சந்தை / கடைத்தெரு. | சந்தை இடம். |
| **Abang Samseng** | "கேங்க்ஸ்டர் சகோதரன்" - தெரு கடினமானவர், சிறு குற்றவாளி. | சட்டவிரோத வேலையைத் தொடங்கும்போது (எ.கா. விலங்கு கடத்தல்) நீங்கள் பேசும் குற்றப் பணி NPC-கள். |
| **Sadaqa** | தர்மம் / பிச்சை அளித்தல் (அரபிலிருந்து, மலாயில் பரவலாகப் பயன்படுத்தப்படுகிறது). | Wallet கருவியின் வீரருக்கு வீரர் நன்கொடை வேண்டுகோளில் உள்ள லேபிள். |
| **Nelayan** | மீனவர். | "Nelayan Sdn. Bhd." என்பது மீன்பிடித்தொழில் தொழிலின் சம்பள வழங்குநர் ("மீனவர் நிறுவனம் வரம்பு" என்று பொருள்). |
| **Sdn. Bhd.** | "Sendirian Berhad"-க்கான சுருக்கம் - "Ltd." அல்லது "Pte Ltd"-க்கு சமமான மலேசிய சொல். | சம்பள அறிவிப்புகளில் தோன்றும். |
| **Jabatan** | துறை / அதிகாரம். | உண்மை உலக பதவித் தலைப்புகளில் பயன்படுத்தப்படுகிறது எ.கா. "Bomba MYSverse" (தீயணைப்பு மற்றும் மீட்புத் துறை). |
| **Kementerian** | அமைச்சகம். | "KKN" என்பது **Kementerian Kesihatan Nasional**-ஐ குறிக்கிறது - Kesihatan வாகனங்களில் உள்ள ஆம்புலன்ஸ் சின்னம். |
| **Jalan** | சாலை / தெரு. | "Jalan Triden", "Jalan Bawang" போன்ற தெரு பெயர்களில் தோன்றும். |
| **Lorong** | ஒழுங்கை / சந்து. | "Lorong Pais", "Lorong Kicai" போன்ற சிறிய தெருக்கள். |

## விளையாட்டு-குறிப்பிட்ட சொற்கள்

| சொல் | வரையறை |
|------|------------|
| **SR** | Sumaya Ringgit - விளையாட்டு உள் நாணய சுருக்கம். "Ringgit" என்பதைப் போன்றது. |
| **Wallet** | உடனடியாக அணுகக்கூடிய உங்கள் பணம். pickpocket மூலம் திருடப்படலாம். |
| **Bank** | உங்கள் பாதுகாப்பான சேமிப்பு. திருடப்பட முடியாது. ATM மூலம் அணுகப்படுகிறது. |
| **Offence points** | குற்றங்களிலிருந்து குவியும் புள்ளிகள். மொத்தம் உங்கள் தேடப்படும் நட்சத்திரங்களைத் தீர்மானிக்கிறது. |
| **Wanted stars** | உங்கள் குற்ற நிலை, 1 முதல் 5 வரை. அதிகம் = மிகவும் தீவிரம். |
| **Wanted decay** | 5 நிமிடங்களுக்கு புதிய குற்றங்கள் எதுவும் செய்யாமல் உங்கள் அனைத்து offence points-களையும் இழப்பது. |
| **Passive mode** | நீங்கள் சேதமடைய முடியாது மற்றும் குற்றங்கள் செய்ய முடியாத ஒரு மாற்றத்தக்க நிலை. |
| **Passive lock** | சில குற்றங்களுக்குப் பிறகு பயன்படுத்தப்படும், passive-mode மாற்றத்தின் தற்காலிக முடக்கம். |
| **LTAA** | "Leave To Avoid Arrest" - குற்றத்தின் நடுவில் வெளியேறுவதன் மூலம் உங்களுக்குக் கிடைக்கும் கூடுதல் சிறை நேரம். |
| **Team** | உங்கள் பாத்திரம் (Visitors, Malaysians, POLIS, BOMBA, Kesihatan). |
| **Sector** | ஒரு வேலை வகை: Fishing, Service, Agro, Bomba, Polis, Kesihatan. |
| **Rank** | ஒரு sector-க்குள் உங்கள் கௌரவ பட்டம் (எ.கா. "Fire Officer"). அலங்காரம் மட்டுமே. |
| **Token** | sector-க்கான உள் பெயர் (எ.கா. "Token_Bomba"). விளையாட்டில் இதை அரிதாகவே பார்ப்பீர்கள். |
| **Plot** | உங்கள் வீட்டைக் கட்ட முடியும் ஒரு அண்டை பகுதியில் ஒரு நிலம். ஒரு வீரருக்கு ஒன்று. |
| **Tier** (வீடு) | வீடு தர நிலை: Tier 1 (மலிவான), Tier 2 (நடுத்தர), Tier 3 (உயர்தர). |
| **Tier** (trawler) | மீன்பிடி படகு தரம்: Tier 1 (இலவசம்), Tier 2 (25k), Tier 3 (37k). |
| **ELS** | Emergency Lighting System - அவசர வாகனங்களில் உள்ள விளக்குகள் மற்றும் சைரன். |
| **LFRT** | Light Fire Rescue Tender - சிறிய தீ வண்டி. |
| **FRT** | Fire Rescue Tender - பெரிய தீ வண்டி. |
| **MPV** | Multi-Purpose Vehicle - சில காவல் கார்களின் உடல் பாணி. |
| **MDT** | Mobile Data Terminal - உங்கள் தொலைபேசியில் காவல்/அவசர அனுப்புதல் செயலி. |
| **999** | மலேசிய அவசர சேவை தொலைபேசி எண் (UK-இல் 911 அல்லது 999 போன்றது). குடிமக்கள் இதை Emergency999 செயலியில் அழைக்கிறார்கள். |
| **Minimap** | உங்கள் பகுதியைக் காட்டும் மேல்-வலது HUD வரைபடம். |
| **Task card** | உங்கள் தற்போதைய நோக்கத்தைக் காட்டும் மேல்-இடது HUD பேனல். |
| **Waypoint** | minimap-இல் ஒரு குறிக்கப்பட்ட இடம். விளையாட்டு-அமைக்கப்பட்டது (பணி, MDT அழைப்பு) அல்லது தனிப்பயன் (வைக்க நீளம்-அழுத்து) ஆக இருக்கலாம். |
| **Guidance beam** | உங்கள் கதாபாத்திரத்திலிருந்து உங்கள் செயலில் உள்ள waypoint-க்கு நோக்கி காட்டும் வண்ண 3D கதிர். |
| **Stud** | Roblox-இன் தூர அலகு. ஏறக்குறைய 1 அடி / 30 செ.மீ. |
| **KKN** | Kementerian Kesihatan Nasional - தேசிய சுகாதார அமைச்சகம். ஆம்புலன்ஸில் காணப்படுகிறது. |
| **Slim Jim** | கார் கடத்தலுக்குப் பயன்படுத்தப்படும் கருவி. ஒரு முறை பயன்பாடு. |
| **Pickpocket Knife** | வீரர்களைப் பாக்கெட்டடிக்கப் பயன்படுத்தப்படும் கருவி. |
| **Stolen plate** | கார் கடத்தப்படும்போது வாகனத்தின் பதிவுத்தகடு குறிக்கப்படுகிறது. POLIS MDT மூலம் தகடுகளைத் தேடலாம். |
| **Paycheck** | ஒரு தொழிலின் பணி-முடிவில் Ringgit கட்டணம், ஒரு அறிவிப்புடன் வழங்கப்படுகிறது. |
| **Injury system** | வீரர்கள் Kesihatan சிகிச்சை தேவைப்படும் கொல்லாத காயங்களை அனுபவிக்க அனுமதிக்கும் மருத்துவ இயக்கவியல். செயலில் இருக்கும்போது, அனைவருக்கும் +15% player-XP கொடுக்கிறது. |
| **TBI** | Traumatic Brain Injury - மிகவும் தீவிரமான காய வகை, படுக்கை ஓய்வு தேவை. |
| **Drop-off zone** | கார் கடத்தல்காரர் திருடப்பட்ட வாகனத்தை விற்க வழங்கும் இடம், அல்லது விலங்கு கடத்தல்காரர்கள் தங்கள் சட்டவிரோத பொருட்களை வழங்கும் இடம். |
| **Nearby officer bonus** | கைது செய்யும் அதிகாரியின் 50 stud-களுக்குள் உள்ள எந்த POLIS வீரருக்கும் கைது செய்யப்பட்ட வெகுமதியில் 50% வழங்கப்படுகிறது. |
| **Combat timer / "ENGAGED"** | taser தாக்கும்போது அல்லது தடுப்பு வெற்றியடையும்போது சந்தேக நபர் மீது திறக்கப்படும் 30-வினாடி ஈடுபாட்டு சாளரம். அது இயங்கும் போது, Reset மற்றும் முதன்மை-மெனு / team-switch பூட்டப்படுகின்றன, மேலும் இறப்பது அல்லது விளையாட்டை விட்டு வெளியேறுவது இரட்டிப்பான கால அளவுள்ள **combat-evasion jail**-க்கு வழிநடத்துகிறது. கைது செய்த அதிகாரி எந்த விதத்திலும் கடனை வைத்திருக்கிறார். [குற்றம் மற்றும் சட்டம்](/sumaya/crime-and-law) ஐப் பார்க்கவும். |
| **Reset cooldown / "Respawning..."** | போரில் இல்லாதபோது நீங்கள் Reset-ஐ அழுத்தும்போது இயங்கும் 12-வினாடி ரத்து செய்யக்கூடிய எண்ணிக்கை. கதாபாத்திரத்தை சுத்தமாக மறுஏற்றுகிறது (இறப்பு இல்லை, கைவிடப்பட்ட கருவிகள் இல்லை). நீங்கள் taser செய்யப்பட்டால், கைது செய்யப்பட்டால், அல்லது விளையாட்டை விட்டு வெளியேறினால் தானாக ரத்து செய்யப்படும். |
| **Citizen level** | உங்கள் உலகளாவிய வீரர் நிலை, மேல்-வலது Status விட்ஜெட்டில் `Citizen <N>` என காட்டப்படுகிறது. sector நிலைகளிலிருந்து வேறுபட்டது (அவை per-தொழில்). |
| **Status widget** | Citizen நிலை + செயலில் உள்ள sector-ஐக் காட்டும் இரண்டு-வரிசை மேல்-வலது HUD பேனல். மேல் பட்டியில் Status ஐகானைத் தட்டுவதன் மூலம் மறைக்கப்பட்டது. |
| **TopBar icons** | திரையின் மேல்-மையத்தில் உள்ள சிறிய Map / Tasks / Status / Open Menu ஐகான்கள். ஒவ்வொரு HUD-ஐயும் மாற்ற தட்டவும்; விளையாட்டின் நடுவில் இறக்காமல் முதன்மை மெனுவை மீண்டும் திறக்க **Open Menu** (வீட்டு குறியீடு) ஐ தட்டவும். |
