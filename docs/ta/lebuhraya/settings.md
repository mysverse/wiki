# Lebuhraya இல் அமைப்புகள் & தனிப்பயனாக்கம்

Lebuhraya உங்கள் விளையாட்டு அனுபவத்தைத் தனிப்பயனாக்கவும், செயல்திறனை மேம்படுத்தவும், உங்கள் தொடர்புகளைக் கட்டுப்படுத்தவும் பல அமைப்புகளை வழங்குகிறது.

## அமைப்புகளை அணுகுதல்

- பெரும்பாலான பொதுவான அமைப்புகளை **முதன்மை மெனு** இல் காணலாம்.
- சில குறிப்பிட்ட மாற்றுகள் அல்லது விருப்பங்கள் விளையாட்டுத் திரையில் உள்ள நிர்ணயிக்கப்பட்ட UI பொத்தான்கள் மூலம் கிடைக்கலாம்.

## செயல்திறன் அமைப்புகள்

இந்த அமைப்புகள் செயல்திறனை மேம்படுத்த உதவும், குறிப்பாக குறைந்த சக்தி கொண்ட சாதனங்களில்.

- ### 🌎 உலகளாவிய நிழல்கள் மாற்று

  - **அறிமுகம்:** v3.3.25
  - **செயல்பாடு:** முழு விளையாட்டிற்கும் நிழல் ரெண்டரிங்கை மாற்ற அனுமதிக்கிறது. நிழல்களை முடக்குவது செயல்திறனை கணிசமாக மேம்படுத்தலாம்.
  - **அணுகல்:** முதன்மை மெனு.

- ### 🌲 மர ரெண்டரிங் மாற்று

  - **அறிமுகம்:** v3.3.15
  - **செயல்பாடு:** மரங்களின் ரெண்டரிங்கை முடக்க வீரர்களை அனுமதிக்கிறது. இது காட்சி குழப்பத்தைக் குறைக்கவும் செயல்திறனை மேம்படுத்தவும் உதவும். விபத்துக்களைக் குறைக்க மரங்கள் நினைவகத்திலும் ஏற்றப்படுகின்றன (ஸ்ட்ரீம் செய்யப்படாது).
  - **அணுகல்:** முதன்மை மெனுவின் மேல் வலது மூலையில் உள்ள பொத்தான்.

- ### ✨ ஒளியமைப்பு தொழில்நுட்பம்
  - **மாற்றம்:** v3.4.21 இல், குறைந்த-நிலை சாதனங்களில் செயல்திறனை மேம்படுத்த ஒளியமைப்பு தொழில்நுட்பம் "Future" இலிருந்து "ShadowMap" ஆக மாற்றப்பட்டது. இது டெவலப்பர்களால் நிர்வகிக்கப்படும் உலகளாவிய அமைப்பாக இருக்கலாம்.

## விளையாட்டு & கட்டுப்பாட்டு அமைப்புகள்

- ### 🏃 தொடுதிரையில் ஓட்டம்

  - **அறிமுகம்:** v3.3.26
  - **செயல்பாடு:** தொடுதிரை சாதனங்களுக்கு ஓட்ட மாற்றுவதை இயக்குகிறது. விசைப்பலகை பயனர்கள் Left Shift ஐப் பயன்படுத்தலாம்.
  - **அணுகல்:** தொடுதல்-நட்பு **⚙️ அமைப்புகள் மெனு**.

- ### 🗺️ விருப்பமான உருவாக்க இடம்

  - **அறிமுகம்:** v3.0.11
  - **செயல்பாடு:** Malaysian மற்றும் Visitor குழுக்களுக்கு, வீரர்கள் Putra Mahkota மற்றும் Jersik இடையே தங்கள் விருப்பமான உருவாக்க இடத்தை அமைக்கலாம்.
  - **அணுகல்:** முதன்மை மெனு.
  - **புதுப்பிப்பு:** v3.5.8 இல் இந்தக் குழுக்களுக்கு இயல்புநிலை உருவாக்க இடம் Putra Mahkota ஆக நிலையாக அமைக்கப்பட்டது, ஆனால் மாற்று இருக்கலாம்.

- ### 📷 MYS Module கேமரா நடத்தை தனிப்பயனாக்கம்

  - **அறிமுகம்:** v3.0.13
  - **செயல்பாடு:** MYS module இல் இயல்புநிலை கேமரா நடத்தையை வீரர்கள் முடக்கலாம் (இந்த நடத்தையின் விவரங்கள் குறிப்பிடப்படவில்லை ஆனால் தனிப்பயன கேமரா ஸ்கிரிப்ட்களைக் குறிக்கலாம்).

- ### 🔘 வாகன மோதல் மாற்று
  - **இடம்:** வாகனத்தில் இருக்கும்போது திரை மீது பொத்தான்.
  - **செயல்பாடு:** வாகன மோதல்களை மாற்ற அனுமதிக்கிறது.
  - **காட்சி மாற்றம் (v3.0.13):** மோதல் பொத்தானின் 'On' நிலை நிறம் பச்சையிலிருந்து இயல்புநிலை கருப்புக்கு மாற்றப்பட்டது.
  - **மொபைல் UI (v3.0.6):** மொபைல் ஓட்டுநர் இடைமுகத்தில் சிறந்த அணுகலுக்காக மோதல் பொத்தான் மறுஅளவிடப்பட்டது.

## UI & இடைமுகம்

- **புதிய UI எழுத்துரு (v3.4.4):** அதிக தெளிவு மற்றும் படிக்கக்கூடிய தன்மைக்காக Roblox இன் "Builder Sans" எழுத்துருவை விளையாட்டு ஏற்றுக்கொண்டது.
- **முதன்மை மெனு அணுகல் (v3.5.9):** உங்கள் கதாபாத்திரத்தை மீட்டமைக்காமல் மேல்பட்டி மூலம் முதன்மை மெனுவை அணுகலாம்.
- **மொபைல் ஓட்டுநர் இடைமுகம் (v3.0.6):** சில பொத்தான்களை மறுஅளவிடுவதன் மூலமும், ஓட்டும்போது கீழ் இடதுபுறத்தில் உள்ள அமைப்புகள் பொத்தானை அகற்றுவதன் மூலமும் குறைந்த குழப்பமான காட்சிக்காக எளிமைப்படுத்தப்பட்டது.
- **Gamepad ஆதரவு (v3.3.27):** gamepad உடன் முதன்மை மெனு விளையாட்டு பொத்தான்கள் தேர்ந்தெடுக்க முடியாத சிக்கலை சரிசெய்தல் உள்ளிட்ட தொடர்ச்சியான மேம்பாடுகள்.

## ஆடியோ அமைப்புகள்

- **MyPod தனிப்பட்ட இசை இயக்கி (v3.0.11):** மேல் பட்டியில் உள்ள இசை ஐகான் மூலம் அணுகவும். உங்களுக்கு மட்டுமே கேட்கக்கூடிய இசையின் தேர்வை கேட்க அனுமதிக்கிறது.
- **சைரன் ஒலி அளவு (v3.4.3):** Kesihatan ஆம்புலன்ஸ் சைரன் ஒலி அளவு குறைக்கப்பட்டது. இது பின்னூட்டத்தின் அடிப்படையில் சில ஆடியோ அளவுகள் டெவலப்பர்களால் உலகளவில் நிர்வகிக்கப்படுவதைக் குறிக்கிறது.

## டெவலப்பர் & நிர்வாகி தொடர்பானவை

- **மேல்நிலை Rank GUI (v3.0.3):** பயனர் பெயர்களுக்குப் பதிலாக வீரர் காட்சிப் பெயர்களைக் காண்பிக்கிறது. வீரர் நிலைகளையும் காண்பிக்கிறது.
- **நிர்வாகி அரட்டை தெரிவுநிலை (v3.3.4):** எளிதான மதிப்பீட்டிற்காக Adonis நிர்வாகிகள் காட்சிப் பெயர்களுக்குப் பதிலாக முழு Roblox பயனர் பெயர்களைப் பார்க்கிறார்கள்.

இந்த அமைப்புகள் உங்கள் Lebuhraya அனுபவத்தின் மீது ஒரு அளவிலான கட்டுப்பாட்டை வழங்குகின்றன. விளையாட்டு புதுப்பிக்கப்படும்போது, புதிய விருப்பங்கள் கிடைக்கலாம். சமீபத்திய அமைப்புகளுக்கு எப்போதும் முதன்மை மெனு மற்றும் திரை UI கூறுகளைப் பாருங்கள்.
