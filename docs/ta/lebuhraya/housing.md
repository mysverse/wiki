# வீடு

லெபுஹ்ராயாவில் ஒரு **குடியிருப்பு அமைப்பு** உள்ளது, அதில் நீங்கள் ஒரு மனையை உரிமையாக்கி, வீட்டை வைத்து, தளபாடங்களால் அலங்கரித்து, உங்கள் சொந்த தளமாகப் பயன்படுத்தலாம். இது விளையாட்டின் மிகவும் சமூக சார்ந்த அம்சங்களில் ஒன்று — நண்பர்களை அழைக்கவும், கதவுகளைப் பூட்டவும், ஒரு கம்புங் பாணி கூடலை நடத்தவும் செய்யலாம்.

## மனையை உரிமையாக்குதல்

பண்டார் ஸெரி புத்ராவில் உள்ள **பங்ஸாபுரி ஸெரி புத்ரா**வில் மனைகள் அமைந்துள்ளன. கிடைக்கக்கூடிய மனைகளின் மொத்த எண்ணிக்கை server-ஆல் அமைக்கப்படுகிறது; அனைத்து மனைகளும் எடுக்கப்பட்டால், **0 available** என்ற குறிப்பானை நீங்கள் காண்பீர்கள்.

1. பங்ஸாபுரி ஸெரி புத்ராவிற்கு ஓட்டிச் செல்லுங்கள்.
2. காலியான மனை குறியிடத்திற்கு நடந்து செல்லுங்கள்.
3. உரிமை கோரல் தூண்டுதலுடன் தொடர்பு கொள்ளுங்கள்.
4. ஒரு வீட்டு வார்ப்புருவைத் தேர்ந்தெடுங்கள் (கீழே காண்க).

உங்கள் மனை உங்கள் கணக்குடன் இணைக்கப்பட்டுள்ளது. server-ஐ விட்டு வெளியேறினாலும் உங்கள் வீடு நிலைத்திருக்கும்; திரும்பி வந்தாலும் அது அங்கேயே இருக்கும்.

## வீட்டு வார்ப்புருக்கள்

ஒரு மனையை உரிமையாக்கும்போது நான்கு வார்ப்புருக்களிலிருந்து ஒன்றை நீங்கள் தேர்ந்தெடுக்கிறீர்கள்:

- **Kampung_House_1** — பாரம்பரிய மலேசிய கம்புங் பாணி வீடு
- **Kampung_House_2** — மற்றொரு கம்புங் வகை, வேறுபட்ட அமைப்பு
- **Kampung_House_3** — மூன்றாவது கம்புங் வகை
- **Modern_Terrace_1** — நவீன டெர்ரேஸ் / டவுன்ஹவுஸ் பாணி

ஒவ்வொரு வார்ப்புருவும் ஒரே உள்ளக பரப்பளவைக் கொண்டுள்ளது ஆனால் வெளிப்புற தன்மை வேறுபடுகிறது. உங்கள் பாத்திர நடிப்பிற்குப் பொருந்துவதைத் தேர்ந்தெடுங்கள்.

## தனிப்பயனாக்குதல்

உங்கள் வீடு வைக்கப்பட்டவுடன், நீங்கள் இதைச் செய்யலாம்:

### பெயிண்ட்

வீட்டின் வெளிப்புற பெயிண்ட் நிறத்தை மாற்றவும்.

### தளபாடங்கள்

தளபாட பட்டியலில் இருந்து பொருட்களைச் சேர்க்கவும். கிடைக்கக்கூடிய பகுதிகள் பின்வருமாறு:

- **Bookshelf**
- **Single Bed**
- **Dining Table**
- **Coffee Table**
- **Chairs**
- **Mirror**

உங்கள் வீட்டுக்குள் பொருட்களை வைக்கவும்; அமைப்பு சரியாக உணரும் வரை அவற்றை நகர்த்தி சுழற்றவும். விளையாட்டின் தற்போதைய பதிப்பில், தளபாட இடும்போது **BR கழிக்கப்படாது** — நாணய அமைப்பு இன்னும் செயல்பாட்டில் இல்லை ([பொருளாதாரம் மற்றும் XP](/lebuhraya/economy#ringgit-br--tracked-but-not-yet-live) காண்க). இலவச அலங்காரம் கிடைக்கும் வரை அதை அனுபவியுங்கள்.

### கதவுகள்

கதவுகளை **பூட்டலாம்** அல்லது **திறக்கலாம்**. பூட்டப்பட்ட கதவு நண்பர்கள் உட்பட மற்ற வீரர்களை வெளியே வைத்திருக்கும் — துணை வேண்டும் என்றால் அதைத் திறக்கவும்.

### விளக்குகள்

உள்ளக விளக்கை ஆன் அல்லது ஆஃப் என மாற்றவும். இரவில் மனநிலையை அமைக்க பயனுள்ளது.

## யார் உள்ளே நுழையலாம்?

- **நீங்கள்** — எப்போதும்.
- **மற்ற வீரர்கள்** — உங்கள் கதவுகள் திறந்திருந்தால் மட்டுமே.
- **குழு உறுப்பினர்கள்** — இயல்பாக சிறப்பு அணுகல் இல்லை; அதே விதிகள் பொருந்தும்.

## வெளியேறுதல் அல்லது மாறுதல்

மற்றொன்றை உரிமையாக்க உங்கள் மனையை கைவிட விரும்பினால், வீட்டு மெனுவின் **release** விருப்பத்தைப் பயன்படுத்தவும். நீங்கள் மனையை விடுவிக்கும்போது உங்கள் தற்போதைய வீடும் தளபாடங்களும் அகற்றப்படும். விடுவிக்கும் முன் உங்கள் அமைப்பு வடிவமைப்பை ஸ்கிரீன்ஷாட்டில் சேமிப்பதைக் கவனியுங்கள் — தானியங்கி மீட்பு எதுவும் இல்லை.

## விலைகள் மற்றும் எண்ணிக்கை குறித்த குறிப்பு

ஒவ்வொரு வீடு மற்றும் தளபாடத்திற்கான சரியான BR விலை, மற்றும் server பக்கத்தில் கிடைக்கும் மொத்த மனைகளின் எண்ணிக்கை, நேரடி விளையாட்டு நிலையைப் பொறுத்தது. இந்த எண்களுக்கு விளையாட்டில் உள்ள மெனுவே எப்போதும் உண்மையின் ஆதாரமாகும்.
