## MYSverse என்றால் என்ன?

MYSverse Roblox-இல் ஆழ்ந்த மலேசிய-கருப்பொருள் அனுபவங்களை உருவாக்குகிறது — மகிழ்ச்சி,
கலாச்சாரம் மற்றும் கல்வி மதிப்புகளை யதார்த்தமான மெட்டாவர்ஸ் அமைப்பில் கலக்கிறது.

## ரோல்பிளே சமூகங்களில் எவ்வாறு சேருவது?

MYSverse Sim-ன் போலீஸ் மற்றும் இராணுவ சமூகங்கள் போன்ற ரோல்பிளே நிறுவனங்கள் பொதுவாக எங்கள் Discord-இல் அறிவிக்கப்படும் ஆட்சேர்ப்புகளை நடத்துகின்றன.
வணிகங்கள் மற்றும் பிற சமூக குழுக்களில் ஈடுபடுவது எப்படி என்பதை அறிய Bandaraya அல்லது Lebuhraya பக்கங்களைப் பார்வையிடவும்.

## கூட்டாண்மைக்கான வாய்ப்புகள் உள்ளனவா?

நிச்சயமாக! நாங்கள் அரசு நிறுவனங்கள் மற்றும் பிராந்திய பிராண்டுகளுடன் ஒத்துழைக்க ஆர்வமாக உள்ளோம்.
தொடர்பு விவரங்களுக்கு எங்கள் [முக்கிய இணையதளத்தை](https://mysver.se) பார்க்கவும்.

## நான் விக்கிக்கு பங்களிக்க முடியுமா?

நிச்சயமாக! எங்கள் விக்கி சமூகத்தால் இயக்கப்படுகிறது மற்றும் GitHub மூலம் திருத்தக்கூடியது.
பக்கத்தின் கீழே உள்ள "இந்தப் பக்கத்தைத் திருத்து" பொத்தானைப் பயன்படுத்தி Pull Request சமர்ப்பிக்கவும் — அதை எங்கள் குழு மதிப்பாய்வு செய்யும்.

## இந்த அனுபவங்களை யார் விளையாட முடியும்?

அனைவருக்கும் வரவேற்பு! எங்கள் உள்ளடக்கம் தனித்துவமாக மலேசியாவாக இருந்தாலும், உலகளாவிய பார்வையாளர்களுக்கு
அணுகக்கூடிய மற்றும் மகிழ்வளிக்கும் அனுபவங்களை நாங்கள் உருவாக்குகிறோம்.
