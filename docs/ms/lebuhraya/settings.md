# Tetapan & Penyesuaian di Lebuhraya

Lebuhraya menawarkan beberapa tetapan untuk menyesuaikan pengalaman permainan anda, mengoptimumkan prestasi, dan mengawal interaksi anda.

## Mengakses Tetapan

- Kebanyakan tetapan umum boleh didapati dalam **Menu Utama**.
- Sesetengah togol atau pilihan tertentu mungkin tersedia melalui butang UI khusus pada skrin permainan.

## Tetapan Prestasi

Tetapan ini boleh membantu meningkatkan prestasi, terutamanya pada peranti yang kurang berkuasa.

- ### 🌎 Togol Bayang Global

  - **Diperkenalkan:** v3.3.25
  - **Fungsi:** Membolehkan anda menogol rendering bayang untuk keseluruhan permainan. Melumpuhkan bayang boleh meningkatkan prestasi dengan ketara.
  - **Akses:** Menu Utama.

- ### 🌲 Togol Rendering Pokok

  - **Diperkenalkan:** v3.3.15
  - **Fungsi:** Membolehkan pemain melumpuhkan rendering pokok. Ini boleh membantu mengurangkan kekusutan visual dan meningkatkan prestasi. Pokok juga dimuatkan dalam memori (bukan distrim) untuk berpotensi mengurangkan kemalangan.
  - **Akses:** Butang di penjuru kanan atas Menu Utama.

- ### ✨ Teknologi Pencahayaan
  - **Perubahan:** Dalam v3.4.21, teknologi pencahayaan ditukar daripada "Future" kepada "ShadowMap" untuk meningkatkan prestasi pada peranti spesifikasi rendah. Ini berkemungkinan tetapan global yang diuruskan oleh pembangun.

## Tetapan Permainan & Kawalan

- ### 🏃 Lari pada Skrin Sentuh

  - **Diperkenalkan:** v3.3.26
  - **Fungsi:** Mengaktifkan togol lari untuk peranti skrin sentuh. Pengguna papan kekunci boleh menggunakan Left Shift.
  - **Akses:** **⚙️ Menu Tetapan** mesra sentuh.

- ### 🗺️ Lokasi Spawn Pilihan

  - **Diperkenalkan:** v3.0.11
  - **Fungsi:** Untuk pasukan Malaysian dan Pelawat, pemain boleh menetapkan lokasi spawn pilihan antara Putra Mahkota dan Jersik.
  - **Akses:** Menu Utama.
  - **Kemas Kini:** Spawn lalai ditetapkan kepada Putra Mahkota secara konsisten untuk pasukan ini dalam v3.5.8, tetapi togol berkemungkinan masih tersedia.

- ### 📷 Penyesuaian Kelakuan Kamera MYS Module

  - **Diperkenalkan:** v3.0.13
  - **Fungsi:** Pemain boleh melumpuhkan kelakuan kamera lalai dalam modul MYS (butiran kelakuan ini tidak dinyatakan tetapi berkemungkinan merujuk kepada skrip kamera tersuai).

- ### 🔘 Togol Perlanggaran Kenderaan
  - **Lokasi:** Butang pada skrin semasa di dalam kenderaan.
  - **Fungsi:** Membolehkan togol perlanggaran kenderaan.
  - **Perubahan Visual (v3.0.13):** Warna keadaan 'On' butang perlanggaran ditukar daripada hijau kepada hitam lalai.
  - **UI Mudah Alih (v3.0.6):** Butang perlanggaran disaiz semula untuk kebolehcapaian lebih baik pada antara muka pemanduan mudah alih.

## UI & Antara Muka

- **Fon UI Baharu (v3.4.4):** Permainan menggunakan fon "Builder Sans" Roblox untuk elemen UI teks bagi kejelasan dan kebolehbacaan yang lebih tinggi.
- **Akses Menu Utama (v3.5.9):** Kini boleh mengakses menu utama melalui topbar tanpa perlu reset watak anda.
- **Antara Muka Pemanduan Mudah Alih (v3.0.6):** Diperkemas dengan menyaiz semula beberapa butang dan membuang butang tetapan dari bahagian kiri bawah semasa memandu untuk paparan yang kurang bersepah.
- **Sokongan Gamepad (v3.3.27):** Penambahbaikan berterusan, termasuk membetulkan butang main menu utama yang tidak boleh dipilih dengan gamepad.

## Tetapan Audio

- **Pemain Muzik Peribadi MyPod (v3.0.11):** Akses melalui ikon muzik di bar atas. Membolehkan anda mendengar pilihan muzik yang hanya anda boleh dengar.
- **Kelantangan Siren (v3.4.3):** Kelantangan siren ambulans Kesihatan dikurangkan. Ini menunjukkan bahawa sesetengah tahap audio diuruskan secara global oleh pembangun berdasarkan maklum balas.

## Berkaitan Pembangun & Pentadbir

- **GUI Pangkat Overhead (v3.0.3):** Memaparkan nama paparan pemain dan bukannya nama pengguna. Juga menunjukkan tahap pemain.
- **Keterlihatan Chat Pentadbir (v3.3.4):** Pentadbir Adonis melihat nama pengguna Roblox penuh dan bukannya nama paparan untuk pengurusan yang lebih mudah.

Tetapan ini memberikan tahap kawalan tertentu ke atas pengalaman Lebuhraya anda. Seiring kemas kini permainan, pilihan baharu mungkin tersedia. Sentiasa semak menu utama dan elemen UI pada skrin untuk tetapan terkini.
