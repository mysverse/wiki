# Glosari

Senarai rujukan perkataan Malaysia (Melayu) dan istilah permainan khusus Sumaya.

## Perkataan Malaysia

Perkataan ini datang daripada kosa kata harian Malaysia/Melayu. Anda akan melihatnya di seluruh permainan.

| Istilah | Maksud | Penggunaan dalam permainan |
|------|---------|------------|
| **Ringgit** | Mata wang Malaysia (dunia sebenar: MYR). | Mata wang dalam permainan Sumaya. Disingkatkan sebagai **SR** dalam UI (Sumaya Ringgit). |
| **Bomba** | Pasukan bomba (daripada Melayu berasal Portugis). | Salah satu pasukan perkhidmatan kecemasan - ahli bomba. |
| **Kesihatan** | Kesihatan / perubatan. | Pasukan perkhidmatan kecemasan perubatan / paramedik. |
| **Polis** | Polis. | Pasukan perkhidmatan kecemasan penguatkuasaan undang-undang. Disingkatkan POLIS. |
| **Jom** | "Mari pergi" (tidak formal). | Butang untuk memasuki dunia daripada menu utama. |
| **Daerah** | Daerah / kawasan. | Kawasan di peta Sumaya. |
| **Mamak** | Restoran Muslim India, sebuah institusi Malaysia. | Salah satu daripada dua tempat sektor perkhidmatan (yang lagi satu ialah makanan segera). |
| **Kedai** | Kedai / gerai. | Papan tanda kedai umum. |
| **Kampung** | Kampung. | Kawasan kediaman di Sumaya; juga nama untuk templat rumah kayu bertiang tradisional (rumah "Borneo" adalah gaya kampung). |
| **Pasaram** | Pasar / bazar. | Lokasi pasar. |
| **Abang Samseng** | "Abang samseng" - seorang gengster jalanan, penjenayah kecil. | NPC misi jenayah yang anda jumpa ketika memulakan kerja haram (cth. pemerdagangan haiwan). |
| **Sadaqa** | Pemberian amal / sedekah (daripada bahasa Arab, digunakan meluas dalam Melayu). | Label pada gesaan derma pemain-ke-pemain pada alat Wallet. |
| **Nelayan** | Nelayan. | "Nelayan Sdn. Bhd." ialah pembekal gaji kerjaya perikanan (bermaksud "Fisherman Co. Ltd."). |
| **Sdn. Bhd.** | Singkatan bagi "Sendirian Berhad" - setara Malaysia dengan "Ltd." atau "Pte Ltd". | Muncul pada notifikasi gaji. |
| **Jabatan** | Jabatan / pihak berkuasa. | Digunakan dalam tajuk pangkat dunia sebenar cth. "Bomba MYSverse". |
| **Kementerian** | Kementerian. | "KKN" bermaksud **Kementerian Kesihatan Nasional** - lambang ambulans pada kenderaan Kesihatan. |
| **Jalan** | Jalan raya / jalan. | Muncul dalam nama jalan seperti "Jalan Triden", "Jalan Bawang". |
| **Lorong** | Lorong. | Jalan-jalan lebih kecil seperti "Lorong Pais", "Lorong Kicai". |

## Istilah khusus permainan

| Istilah | Definisi |
|------|------------|
| **SR** | Sumaya Ringgit - singkatan mata wang dalam permainan. Sama seperti "Ringgit". |
| **Wallet** | Wang tunai yang boleh diakses serta-merta. Boleh dicuri melalui pickpocket. |
| **Bank** | Simpanan selamat anda. Tidak boleh diseluk saku. Diakses melalui ATM. |
| **Offence points** | Mata yang terkumpul daripada jenayah. Jumlah menentukan bintang buruan anda. |
| **Wanted stars** | Status jenayah anda, 1 hingga 5. Lebih tinggi = lebih teruk. |
| **Wanted decay** | Kehilangan semua offence points dengan tidak melakukan jenayah baharu selama 5 minit. |
| **Passive mode** | Keadaan boleh-tukar yang menjadikan anda tidak boleh dicederakan dan tidak boleh melakukan jenayah. |
| **Passive lock** | Penyahdayaan sementara pada penukaran passive mode, dikenakan selepas kesalahan tertentu. |
| **LTAA** | "Leave To Avoid Arrest" - masa penjara tambahan yang anda terima kerana log keluar semasa kesalahan. |
| **Team** | Peranan anda (Visitors, Malaysians, POLIS, BOMBA, Kesihatan). |
| **Sector** | Jenis kerja: Fishing, Service, Agro, Bomba, Polis, Kesihatan. |
| **Rank** | Gelaran prestij anda dalam sektor (cth. "Fire Officer"). Hanya kosmetik. |
| **Token** | Nama dalaman untuk sektor (cth. "Token_Bomba"). Anda jarang melihatnya dalam permainan. |
| **Plot** | Sebidang tanah di kejiranan di mana anda boleh bina rumah. Satu bagi setiap pemain. |
| **Tier** (rumah) | Tahap kualiti rumah: Tier 1 (murah), Tier 2 (pertengahan), Tier 3 (premium). |
| **Tier** (trawler) | Kualiti bot menangkap ikan: Tier 1 (percuma), Tier 2 (25k), Tier 3 (37k). |
| **ELS** | Emergency Lighting System - lampu dan siren pada kenderaan kecemasan. |
| **LFRT** | Light Fire Rescue Tender - trak bomba yang lebih kecil. |
| **FRT** | Fire Rescue Tender - trak bomba yang lebih berat. |
| **MPV** | Multi-Purpose Vehicle - gaya badan sesetengah kereta polis. |
| **MDT** | Mobile Data Terminal - aplikasi penghantaran polis/kecemasan pada telefon anda. |
| **999** | Nombor telefon perkhidmatan kecemasan Malaysia (seperti 911 atau 999 di UK). Orang awam mendailnya dalam aplikasi Emergency999. |
| **Minimap** | Peta HUD di kanan atas yang memaparkan kawasan anda. |
| **Task card** | Panel HUD di kiri atas yang memaparkan objektif semasa anda. |
| **Waypoint** | Lokasi bertanda pada minimap. Boleh ditetapkan oleh permainan (misi, panggilan MDT) atau tersuai (tekan lama untuk letak). |
| **Guidance beam** | Pancaran 3D berwarna yang menunjuk daripada watak anda ke waypoint aktif anda. |
| **Stud** | Unit jarak Roblox. Lebih kurang 1 kaki / 30 cm. |
| **KKN** | Kementerian Kesihatan Nasional. Dilihat pada ambulans. |
| **Slim Jim** | Alat yang digunakan untuk merampas kereta. Satu kali guna. |
| **Pickpocket Knife** | Alat yang digunakan untuk menyeluk saku pemain. |
| **Stolen plate** | Plat kenderaan ditandakan apabila dirampas. POLIS boleh menyemak plat melalui MDT. |
| **Paycheck** | Bayaran Ringgit hujung syif daripada kerjaya, dihantar bersama notifikasi. |
| **Injury system** | Mekanik perubatan yang membolehkan pemain mengalami kecederaan bukan maut yang memerlukan rawatan Kesihatan. Apabila aktif, memberi semua orang +15% XP pemain. |
| **TBI** | Traumatic Brain Injury - jenis kecederaan paling serius, memerlukan rehat di katil. |
| **Drop-off zone** | Lokasi di mana perampas kereta menghantar kenderaan curian untuk dijual, atau di mana pemerdagang haiwan menghantar barang haram mereka. |
| **Nearby officer bonus** | 50% ganjaran tangkapan diberikan kepada mana-mana pemain POLIS dalam lingkungan 50 studs daripada pegawai yang menangkap. |
