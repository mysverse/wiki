# Glosari

Senarai rujukan perkataan Malaysia (Melayu) dan istilah permainan khusus Sumaya.

## Perkataan Malaysia

Perkataan ini berasal daripada kosa kata harian Malaysia/Melayu. Anda akan melihatnya di seluruh permainan.

| Istilah | Maksud | Kegunaan dalam permainan |
|------|---------|------------|
| **Ringgit** | Mata wang Malaysia (dunia sebenar: MYR). | Mata wang dalam permainan Sumaya. Disingkatkan sebagai **SR** dalam UI (Sumaya Ringgit). |
| **Bomba** | Pasukan bomba (daripada perkataan Melayu berasal Portugis). | Salah satu pasukan perkhidmatan kecemasan - ahli bomba. |
| **Kesihatan** | Kesihatan / perubatan. | Pasukan perkhidmatan kecemasan perubatan / paramedik. |
| **Polis** | Polis. | Pasukan perkhidmatan kecemasan penguatkuasaan undang-undang. Disingkatkan POLIS. |
| **Jom** | "Mari pergi" (tidak formal). | Butang untuk masuk ke dunia permainan dari menu utama. |
| **Jelajah** | "Meneroka" / "perjalanan". | Aplikasi jurnal penemuan pada telefon — menjejak mercu tanda (POI) yang anda telah jumpa berbanding yang masih terkunci. |
| **Ditemui** | "Dijumpai" / "ditemukan". | Digunakan dalam pil kemajuan aplikasi Jelajah (`<n> / <total> ditemui`). |
| **Baca Maklumat** | "Baca maklumat". | Butang CTA pada skrin terperinci POI Jelajah yang telah ditemui — membuka semula infografik mercu tanda. |
| **Belum** | "Belum lagi". | Pil penapis dalam Jelajah untuk mercu tanda yang belum anda temui. |
| **Semua** | "Semua". | Pil penapis dalam Jelajah untuk senarai POI penuh. |
| **Daerah** | Daerah / wilayah. | Wilayah peta Sumaya. |
| **Mamak** | Restoran Muslim India, satu institusi Malaysia. | Salah satu daripada dua tempat sektor perkhidmatan (satu lagi makanan segera). |
| **Kedai** | Kedai. | Papan tanda kedai umum. |
| **Kampung** | Kampung. | Kawasan kediaman di Sumaya; juga nama untuk templat rumah kayu bertiang tradisional (rumah "Bornean" adalah gaya kampung). |
| **Pasaram** | Pasar / bazar. | Lokasi pasaraya. |
| **Abang Samseng** | "Abang gengster" - lelaki kasar jalanan, penjenayah kecil. | NPC misi jenayah yang anda berbual dengannya semasa memulakan kerja haram (cth. pemerdagangan haiwan). |
| **Sadaqa** | Pemberian sedekah / amal jariah (daripada bahasa Arab, digunakan secara meluas dalam bahasa Melayu). | Label pada gesaan derma pemain ke pemain pada alat Wallet. |
| **Nelayan** | Nelayan. | "Nelayan Sdn. Bhd." adalah pembekal gaji kerjaya menangkap ikan (bermaksud "Syarikat Nelayan Berhad"). |
| **Sdn. Bhd.** | Singkatan untuk "Sendirian Berhad" - bersamaan Malaysia dengan "Ltd." atau "Pte Ltd". | Muncul pada notifikasi gaji. |
| **Jabatan** | Jabatan / pihak berkuasa. | Digunakan dalam tajuk pangkat dunia sebenar cth. "Bomba MYSverse" (Jabatan Bomba dan Penyelamat). |
| **Kementerian** | Kementerian. | "KKN" bermaksud **Kementerian Kesihatan Nasional** - lambang ambulans pada kenderaan Kesihatan. |
| **Jalan** | Jalan. | Muncul dalam nama jalan seperti "Jalan Triden", "Jalan Bawang". |
| **Lorong** | Lorong. | Jalan-jalan yang lebih kecil seperti "Lorong Pais", "Lorong Kicai". |

## Istilah khusus permainan

| Istilah | Definisi |
|------|------------|
| **SR** | Sumaya Ringgit - singkatan mata wang dalam permainan. Sama seperti "Ringgit". |
| **Wallet** | Wang tunai anda yang boleh diakses serta-merta. Boleh dicuri melalui pickpocket. |
| **Bank** | Simpanan selamat anda. Tidak boleh dicuri. Diakses melalui ATM. |
| **Offence points** | Mata yang terkumpul daripada jenayah. Jumlah menentukan bintang dikehendaki anda. |
| **Wanted stars** | Status jenayah anda, 1 hingga 5. Lebih tinggi = lebih serius. |
| **Wanted decay** | Kehilangan semua offence points anda dengan tidak melakukan jenayah baharu selama 5 minit. |
| **Passive mode** | Keadaan boleh-tukar di mana anda tidak boleh dicederakan dan tidak boleh melakukan jenayah. |
| **Passive lock** | Penyahaktifan sementara pertukaran passive-mode, dikenakan selepas kesalahan tertentu. |
| **LTAA** | "Leave To Avoid Arrest" - masa penjara tambahan yang anda dapat kerana log keluar di tengah-tengah kesalahan. |
| **Team** | Peranan anda (Visitors, Malaysians, POLIS, BOMBA, Kesihatan). |
| **Sector** | Jenis pekerjaan: Fishing, Service, Agro, Bomba, Polis, Kesihatan. |
| **Rank** | Tajuk prestij anda dalam satu sektor (cth. "Fire Officer"). Hanya kosmetik. |
| **Token** | Nama dalaman untuk sektor (cth. "Token_Bomba"). Anda jarang melihatnya dalam permainan. |
| **Plot** | Sekeping tanah dalam kejiranan di mana anda boleh membina rumah. Satu per pemain. |
| **Tier** (rumah) | Tahap kualiti rumah: Tier 1 (murah), Tier 2 (sederhana), Tier 3 (premium). |
| **Tier** (bot tunda) | Kualiti bot menangkap ikan: Tier 1 (percuma), Tier 2 (25k), Tier 3 (37k). |
| **ELS** | Emergency Lighting System - lampu dan siren pada kenderaan kecemasan. |
| **LFRT** | Light Fire Rescue Tender - lori bomba yang lebih kecil. |
| **FRT** | Fire Rescue Tender - lori bomba yang lebih berat. |
| **MPV** | Multi-Purpose Vehicle - gaya badan beberapa kereta polis. |
| **MDT** | Mobile Data Terminal - aplikasi penghantar polis/kecemasan pada telefon anda. |
| **999** | Nombor telefon perkhidmatan kecemasan Malaysia (seperti 911 atau 999 di UK). Orang awam mendailnya dalam aplikasi Emergency999. |
| **Minimap** | Peta HUD kanan atas yang menunjukkan kawasan anda. |
| **Task card** | Panel HUD kiri atas yang menunjukkan objektif semasa anda. |
| **Waypoint** | Lokasi yang ditandakan pada minimap. Boleh ditetapkan permainan (misi, panggilan MDT) atau tersuai (tekan-lama untuk meletakkan). |
| **Guidance beam** | Pancaran 3D berwarna yang menunjuk dari watak anda ke waypoint aktif. |
| **Stud** | Unit jarak Roblox. Lebih kurang 1 kaki / 30 cm. |
| **KKN** | Kementerian Kesihatan Nasional. Dilihat pada ambulans. |
| **Slim Jim** | Alat yang digunakan untuk merampas kereta. Sekali guna. |
| **Pickpocket Knife** | Alat yang digunakan untuk mencuri saku pemain. |
| **Stolen plate** | Plat kenderaan ditandakan apabila dirampas. POLIS boleh menyemak plat melalui MDT. |
| **Paycheck** | Bayaran Ringgit hujung syif daripada kerjaya, dihantar dengan notifikasi. |
| **Injury system** | Mekanik perubatan yang membenarkan pemain mengalami kecederaan tidak membawa maut yang memerlukan rawatan Kesihatan. Apabila aktif, memberi semua orang +15% player-XP. |
| **TBI** | Traumatic Brain Injury - jenis kecederaan paling serius, memerlukan rehat di katil. |
| **Drop-off zone** | Lokasi di mana perampas kereta menghantar kenderaan yang dicuri untuk dijual, atau di mana penyeludup haiwan menghantar barang haram mereka. |
| **Nearby officer bonus** | 50% daripada ganjaran tangkapan diberikan kepada mana-mana pemain POLIS dalam lingkungan 50 stud daripada pegawai yang menangkap. |
| **Combat timer / "ENGAGED"** | Tetingkap penglibatan 30-saat yang terbuka pada suspek apabila taser kena atau penahanan berjaya. Semasa ia berjalan, Reset dan menu utama / pertukaran pasukan dikunci, dan mati atau meninggalkan permainan akan menghala ke **combat-evasion jail** yang berdurasi gandaan. Pegawai yang menangkap mengekalkan kredit dalam kedua-dua keadaan. Lihat [Jenayah dan Undang-Undang](/sumaya/crime-and-law). |
| **Reset cooldown / "Respawning..."** | Kiraan undur 12-saat yang boleh dibatalkan yang berjalan apabila anda menekan Reset di luar pertempuran. Memuat semula watak dengan bersih (tiada kematian, tiada alat dijatuhkan). Dibatalkan secara automatik jika anda kena taser, ditangkap, atau meninggalkan permainan. |
| **Citizen level** | Tahap pemain global anda, ditunjukkan pada widget Status kanan atas sebagai `Citizen <N>`. Berbeza daripada tahap sektor (yang adalah per-kerjaya). |
| **Status widget** | Panel HUD kanan atas dua-baris yang menunjukkan tahap Citizen + sektor aktif. Disembunyikan dengan mengetik ikon Status pada bar atas. |
| **TopBar icons** | Ikon-ikon kecil Map / Tasks / Status / Open Menu di tengah-atas skrin. Ketik untuk menukar setiap HUD; ketik **Open Menu** (glif rumah) untuk membuka semula menu utama di tengah-tengah permainan tanpa mati. |

## Singkatan yang biasa dilihat

| Singkatan | Maksud |
|--------|---------|
| SR | Sumaya Ringgit (mata wang) |
| XP | Mata pengalaman |
| POI | Tempat Menarik (Point of Interest, pada minimap) |
| HUD | Paparan Atas Kepala (Heads-Up Display) |
| NPC | Watak Bukan Pemain (Non-Player Character) |
| PvP | Pemain lawan Pemain |
| ATM | Mesin Juruwang Automatik |
